# 夜更けニュース (yofuke-news)

> 深夜 3 時、 ターミナル で `yofuke-news` と 叩く と、 今夜 の indie ゲーム ニュース 5 件 が ASCII カード で 流れる。 各 カード に は 「あなた の 睡眠 と 引き換え に 5.4 時間 の プレイ」 「クリア まで に 失う 朝食 12 回」 の ような 絶妙 に バカっぽい 換算 が 添えられる。

夜型 ゲーマー が 深夜 に カチカチ ターミナル を 開く 文化 と、 ゲーミング ニュース と、 「睡眠 を 失う 計算」 の 完全 な 非接続 が 笑い を 生む。 streak も 「夜更かし 注意」 も 出さない、 ただ 一緒 に 夜更かし してくれる CLI。

## 何 が 出る か

```text
─────────────────────────────────────────────────────────────
 ✦ INDIE  RELEASES  (2026-05-17, 02:14 JST)                   
─────────────────────────────────────────────────────────────

  ┌─ 01 ─────────────────────────────────────────────────┐
  │  Hollow Frequencies                                  │
  │  ロックステディ系 メトロイドヴァニア、 ¥1,980          │
  │                                                       │
  │  プレイ時間目安 : 12 時間                              │
  │  あなた の 睡眠 と 引き換え に : 4.8 時間            │
  │  クリア まで に 失う 朝食 : 12 回                      │
  └───────────────────────────────────────────────────────┘

  ┌─ 02 ─────────────────────────────────────────────────┐
  ...
```

各 ニュース は static データ (2026 年 5 月 中旬 を 想定 した curated + 創作)。

## 走らせる

```bash
cd yofuke-news
npm install

# 直接 実行
node src/cli.js

# 全画面 (5 件 全部)
node src/cli.js --all

# テスト
npm test
```

`npm install -g .` で `yofuke-news` を どこ から でも 呼べる。

## 哲学

- 「夜更かし 注意!」 「もう 寝なさい」 のような 命令 / 罪悪感 は 一切 出さない (cycle 20 kyuufu の 隣 で 暮らす トーン)
- 換算 は 絶妙 に バカっぽい、 でも 攻撃 的 で は ない (「廃人」 「ダメ人間」 禁止)
- 数字 は 創作、 ゲーム タイトル も 創作 (実在 風)、 ただし 「2026 年 5 月 中旬 を 想定」 と 明記
- 1 回 の 実行 で 5 件 だけ、 スクロール 不要、 30 秒 で 読了
