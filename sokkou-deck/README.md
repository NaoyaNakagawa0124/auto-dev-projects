# 速攻デッキ (sokkou-deck)

> 効率厨 ゲーマー の ため の、 スピードラン 形式 フラッシュ カード トレーナー。

フラッシュカード を 「タイム アタック」 として 解く。 デッキ を 全部 答え終わる まで の ms を 記録、 PB (Personal Best) と 比較、 「フレーム データ」 で どの カード で 何 ms 詰まった か を 振り返る。 暗記 を ゲーム の ように 詰める。

```
┌─ DECK: TOEIC 600 単語 (50 cards) ─────────────────────────────┐
│ MODE: any%        TIMER  00:32.184      PB  00:31.502 (-682)  │
│                                                                │
│         ┌──────────────────────────────────────┐               │
│         │            diligent                  │               │
│         │  ──────────────────────────────────  │               │
│         │  > 勤勉な_                            │               │
│         └──────────────────────────────────────┘               │
│                                                                │
│  SPLITS                                                        │
│  01  abundant      豊富な          412ms   -23                 │
│  02  brittle       脆い            389ms   +12                 │
│  03  candid        率直な          451ms   -8                  │
│  04  diligent      勤勉な          ...                         │
└────────────────────────────────────────────────────────────────┘
```

## なぜ Rust + WASM か

- 入力 から 判定 まで の レイテンシ を 1 ms 単位 で 制御 し たい (任意 の ゲーム ロジック を ブラウザ で)
- localStorage に PB を 保存、 オフライン で 動く、 サーバー 不要
- カード ロジック を Rust で 書く と JSON シリアライズ も 型安全

## 機能

### 3 つ の モード
- **any%** — 全カード を できる だけ 速く。 ミス OK (どんな 答え でも 次 へ)
- **100%** — 全カード を 正解 で。 間違える と 5 秒 ペナルティ
- **コンシステンシー** — 10 問 連続 正解、 最速 タイム を 目指す

### 内蔵 デッキ (5 種)
- 🇬🇧 **TOEIC 600 単語** — 英単語 → 日本語 (50 枚)
- 💻 **IT パスポート 用語** — IT 用語 → 意味 (40 枚)
- ⚗️ **元素 記号** — 記号 → 元素名 (50 枚)
- 📚 **JLPT N1 慣用句** — 慣用句 → 意味 (30 枚)
- 🏛️ **歴代 総理大臣** — 名前 → 在任 期間 (30 枚)

### フレーム データ
ラン 終了 後、 1 枚 ずつ の 反応 時間 を 表 で 見せる。 PB と の 差分 (delta) を 色 で。 「どこ で 詰まった か」 が ms 単位 で 分かる。

### ゴースト ペース
ラン 中、 過去 PB の 同 タイミング を 半透明 で 表示。 「PB 比 +234 ms」 が リアルタイム で 見える。

## ビルド

```bash
# Rust → WASM
wasm-pack build --target web --out-dir www/pkg

# ローカル サーバー
cd www && python3 -m http.server 8000
# → http://localhost:8000
```

## テスト

```bash
cargo test
```

純ロジック (deck / run / stats / 入力 正規化) を `cargo test` で。 UI 操作 は test し ない。

## データ 保存

`localStorage` に:
- `sokkou-deck:pb` — デッキ × モード ごと の PB
- `sokkou-deck:settings` — テーマ / 効果 音 ON/OFF / 直近 デッキ
