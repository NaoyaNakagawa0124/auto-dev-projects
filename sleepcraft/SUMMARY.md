# SleepCraft — Build Summary

## What Was Built

A Dart package (Flutter-ready) that looks like a sleep tracker but secretly teaches real CBT-I (Cognitive Behavioral Therapy for Insomnia) techniques. Five breathing exercises, 14 micro-lessons, sleep scoring, streak tracking, and sleep window calculations — all disguised as "helpful tips."

## The Secret Curriculum

- **Breathing exercises** teach 4-7-8 breathing, box breathing, PMR, diaphragmatic breathing, and Chandra Bhedana
- **Micro-lessons** teach sleep hygiene, stimulus control, cognitive restructuring, and circadian rhythm management
- **Sleep window** implements sleep restriction therapy (the most effective CBT-I component)
- **Scoring** tracks sleep efficiency — the real metric therapists use

Users think they're using a sleep tracker. They're actually completing a structured CBT-I program.
