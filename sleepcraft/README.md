# SleepCraft

> A sleep tracker that secretly teaches you how to sleep better.

## What is SleepCraft?

SleepCraft looks like a simple sleep log. But hidden inside every feature is a real CBT-I technique — the gold standard for treating sleep problems.

**You think you're tracking sleep. You're actually learning breathing, sleep hygiene, and relaxation skills.**

## Features

| Feature | What you see | Secret skill taught |
|---------|-------------|---------------------|
| Wind-Down Timer | Countdown to bed | **4-7-8 breathing** |
| Sleep Window | Bedtime/wake schedule | **Sleep restriction therapy** |
| Thought Journal | Brain dump before bed | **Cognitive restructuring** |
| Sleep Score | Quality rating | **Sleep efficiency tracking** |
| Streak Calendar | Consistency view | **Stimulus control** |

## Quick Start

```bash
cd sleepcraft
dart pub get
dart test
dart run example/sleepcraft_example.dart
```
