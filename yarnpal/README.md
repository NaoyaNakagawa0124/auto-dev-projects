# YarnPal 🧶

> Your knitting companion. Big buttons. Simple counting. Happy grandparent.

## What is YarnPal?

YarnPal is a mobile PWA designed for grandparents (or anyone!) who knit or crochet. It has three screens with huge, easy-to-tap buttons:

1. **Counter** — Tap to count rows and stitches. Big numbers. Undo button.
2. **Projects** — Keep a list of what you're making with yarn color and needle size.
3. **Timer** — Simple knitting session timer.

No accounts. No cloud. No tiny text. Just you and your yarn.

## Install

```bash
python3 -m http.server 8080
# Open on phone → "Add to Home Screen"
```
