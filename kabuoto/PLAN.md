# Kabuoto — Implementation Plan

## Phase 1: Core Audio Engine & Data
- Web Audio API synthesizer with multiple oscillator types
- Stock data model and sample datasets (real historical data)
- Price → pitch mapping, volatility → tempo, volume → amplitude

## Phase 2: Visualization
- Canvas-based price chart synced to audio playback
- Audio waveform visualization (oscilloscope style)
- Playback position indicator on chart

## Phase 3: UI & Controls
- Stock ticker selection (sample data for major stocks)
- Playback controls (play/pause/speed)
- Sound parameter tuning (scale, instrument, tempo range)
- Beautiful dark financial-meets-music UI

## Phase 4: Polish
- Japanese text throughout
- Multiple musical scales (major, minor, pentatonic, Japanese)
- Responsive layout
- Screenshot-worthy design
