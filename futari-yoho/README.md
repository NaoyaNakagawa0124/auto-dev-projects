# 二人予報 (futari-yoho)

> 共働き夫婦のための、判定しない気分予報

すれ違いがちな二人が、毎晩 30 秒だけ画面の前で「今日どうだった?」を残せる場所。
気分も体力も「一人になりたい度」も、ただ天気アイコンに翻訳されるだけ。
「もっと話そう」とは言わない。「並んでいる夜だね」と、そっと教えてくれる。

## Why

共働き夫婦の悩みのひとつは「お互いの状態がわからない」こと。

- 「今夜は話したい?それとも一人時間が欲しい?」を毎回口に出すのは疲れる
- かといってスプレッドシートでお互いを評価するのは絶対に嫌
- 既存の「カップル管理アプリ」は明るすぎ・ロマンチックすぎて続かない

判定もスコアも進捗ゲージもない。ただ「今夜のふたりの空模様」を見て、
「あ、二人とも雨だ。じゃあ並んで黙ろう」と思える、その静けさが欲しかった。

## What

ターミナルで動く Textual TUI。

- 一日一回、4 つの質問にタップだけで答える (30 秒)
- 気分 / 体力 / 一人時間が欲しい度 / 一言メモ
- 今日の二人を「天気」として並べて表示
- 直近 7 日間を「ふたりの空のグリッド」として眺める
- 今夜のおすすめモード (話す夜 / 並んで居る夜 / ひとりひとりの夜) を、
  指示せずに *呟く*

## Tech Stack

- Python 3.11+
- Textual 8.x (TUI)
- Rich (色とコンポジット)
- JSON ファイルで永続化 (`~/.futari-yoho/data.json`)
- 依存は最小限

## Install / Run

```bash
cd futari-yoho
pip install -e .
futari-yoho
```

## Design Principle

- **判定しない** — スコア・ランキング・進捗バーは置かない
- **静か** — 派手な色やアニメーションを避け、夜の照明のように
- **30 秒で終わる** — 続かなければ意味がない
- **二人の不在を許す** — どちらかが check-in しなくても画面は壊れない
