# Tenkimeshi — Implementation Plan

## Phase 1: Core Python Logic
- Weather fetching from OpenMeteo API (free, no key)
- Food recommendation engine (weather → food mapping)
- Restaurant floor plan ASCII generator
- Data models and configuration

## Phase 2: RPi Display Output
- Terminal display formatter (simulates LED/e-ink)
- Box-drawing UI for terminal output
- Color-coded weather/food display
- Auto-refresh loop for continuous display

## Phase 3: Web Simulator
- Single HTML file simulator showing device mockup
- Fetches weather via JS fetch API
- Renders food recommendation + floor plan
- Beautiful IoT device frame visual

## Phase 4: Documentation & Tests
- Wiring guide (Markdown with ASCII diagrams)
- Hardware BOM (bill of materials)
- Python tests for recommendation engine
- Japanese text throughout
