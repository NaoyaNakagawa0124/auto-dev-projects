# クイズノート (KuizuNote) — 脳トレパズルノートブック

シニアのための、美しいビジュアルパズル。Jupyter Notebookで楽しく脳トレ！

## 特徴

- **10問のパズル** — パターン認識、数列、グラフ読解、論理パズル
- **美しいグラフ** — matplotlib で描かれた視覚的なパズル
- **脳タイプ診断** — 最後にあなたの脳タイプを診断
- **インタラクティブ** — セルを実行して回答を入力

## 実行方法

```bash
pip install matplotlib numpy
jupyter notebook kuizunote.ipynb
```

## Tech Stack

- Jupyter Notebook
- Python / matplotlib / numpy
