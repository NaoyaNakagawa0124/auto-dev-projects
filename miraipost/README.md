# 未来ポスト (Miraipost) - 未来の自分への手紙

A Chrome extension for writing letters to your future self. Seal your words, choose a delivery date, and receive them back as gentle browser notifications when the time comes.

## Features

- **手紙を書く**: Write letters to your future self with a warm, handwritten-style editor
- **配達日を選ぶ**: Choose when your letter will be delivered (1 week, 1 month, 3 months, 1 year, custom)
- **不安ポスト**: Write your current worries, seal them, and revisit later with fresh perspective
- **手紙が届く**: Receive sealed letters as gentle notifications on the chosen date
- **手紙一覧**: Browse delivered and pending letters in a beautiful letterbox UI
- **封蝋デザイン**: Sealed letters with wax seal visual — break the seal to read

## Tech Stack

- Chrome Extension Manifest V3
- Vanilla JS / HTML / CSS
- Chrome Storage API (local)
- Chrome Alarms API (delivery scheduling)
- Chrome Notifications API

## Install

1. Open `chrome://extensions/`
2. Enable "Developer mode"
3. Click "Load unpacked" and select the `miraipost/` directory

## Design Philosophy

Warm, paper-like textures. Handwritten-style fonts. Wax seals. Everything feels like receiving a real letter — not a digital notification. Designed to be a moment of calm in a busy browser.
