# 両替勘 (ryogae-kan)

> 海外旅行中の値段感覚を鍛える、5秒勝負のWebゲーム。Rust+WASMで動く。

## これは何？

**両替勘** は、海外一人旅で「この値段、安い？高い？」と一瞬で判断する勘を鍛えるスピードゲームです。

- 異国の屋台や売店が値段を提示してくる: 「€38」「₹4,500」「₩12,000」「฿420」「Rp 78,000」…
- あなたは 5 秒以内に「**安い / 妥当 / 高い**」を判定する
- 連続正解でストリークが伸び、難しい通貨が解放される
- 円換算と参考価格が答え合わせで表示される

電卓を叩いていては勝てません。指が覚えるまでひたすら遊ぶ。

## 特徴

- **15 通貨 × 10 品目**: 主要通貨と現地通貨を網羅。コーヒー / 水 / 屋台メシ / タクシー / ホテルなど身近な品目
- **エスカレート**: ストリークに応じて制限時間が縮み (5秒 → 4秒 → 3秒) 通貨難度も上がる
- **Rust+WASM コア**: 計算とゲーム状態は Rust。HTML はだけ薄い表示層
- **パスポートスタンプ意匠**: 航空券・スタンプ・搭乗券のデザイン語彙
- **ローカル保存**: 最高ストリーク・最高スコアを localStorage に保存
- **完全日本語UI**

## 動かし方

```bash
# Rust + wasm-pack を用意 (https://rustwasm.github.io/wasm-pack/installer/)
cd ryogae-kan
wasm-pack build --target web --out-dir www/pkg --release

# あるいは
make build

# 起動
cd www && python3 -m http.server 8000
# http://localhost:8000
```

### テスト

```bash
cargo test
```

## 操作

- 中央に表示された価格と品目を見る
- 「**🟢 安い**」「**🟡 妥当**」「**🔴 高い**」のどれかを 5 秒以内にタップ／クリック
- 正解で +1 ストリーク。間違いでゲームオーバー
- 数字キー 1/2/3 でも判定可能
