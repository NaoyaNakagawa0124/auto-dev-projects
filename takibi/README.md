# 焚火 (takibi)

> カフェで作業するノマドのための、ポケットの中の小さな焚火。

## これは何？

スマホの画面の中で、ピクセルアートの焚火が静かに燃え続けます。
集中して作業している間は、焚火は穏やかにパチパチと鳴り、
あなたが「ちょっと休もう」と思った時、火は明るく揺らめいて、
そっと一息つくことを誘います。

ログインなし。記録なし。目標なし。
ただ静かに、隣で燃え続ける焚火だけがあります。

## 着想

『Dark Souls』のかがり火、『Hollow Knight』のベンチ、『Spiritfarer』の火。
インディーゲームには「セーブポイント＝休む場所」という静かな伝統があります。
カフェで一人作業するノマドにも、そんな「ひとやすみの場所」があっていい。

## 機能

- **ピクセルアート焚火**: Canvas で描かれる、ゆらめく炎と舞う火の粉
- **環境音**: Web Audio API で生成される、薪のはぜる音と風の音
- **集中タイマー**: 25分集中 → 5分休憩のポモドーロ（無音通知）
- **「火に薪をくべる」**: 集中サイクルを完了すると、火が少し大きくなる
- **オフライン対応**: PWA としてホーム画面に追加可能

## 使い方

```
open index.html
```

ブラウザで開くだけ。サーバー不要。

スマホで使う場合は、ローカル HTTP サーバー経由でアクセスし、
「ホーム画面に追加」で PWA としてインストールできます。

```
python3 -m http.server 8000
# http://<your-ip>:8000 にスマホからアクセス
```

## 技術スタック

- HTML / CSS / Vanilla JavaScript
- Canvas 2D（焚火アニメーション）
- Web Audio API（環境音生成）
- LocalStorage（セッション記録）
- Service Worker（PWA / オフライン対応）
