# ふたり暦 (futari-koyomi)

> 共働き 夫婦 の ための 「今宵、 ふたり で 何 を するか」 ダッシュボード。

朝 7 時 に 出て 夜 9 時 に 帰ってくる 2 人 で、 ふと 顔 を 合わせる 30 分。 ふたり暦 は 「今日 は 何 の 日」 を 1 枚 の カード に 出し、 3 つ の 「5 分 で 終わる 一緒 の 何か」 を 提案 する。 タップ で 「今夜 の 約束」 が 確定、 やった と 言って 「やった ね」 を 押す と 2 人 の 暦 に 1 行 残る。 1 年 経つ と、 ふたり が 過ごした 365 個 の 小さな 「日」 が 並ぶ。

```
─ 2026-05-17  日曜 ─────────────────────────────────────────
  ✦  今日 は 「世界 子守唄 の 日」                     #57 / 365
  
  共働き の ふたり が 今宵 5 分 で 一緒 に できる こと:

  ①  好き な 子守唄 を YouTube で 1 曲、 並んで 聴く
  ②  もし 子供 が いたら 何 を 歌う か、 5 分 だけ 話す
  ③  自分 が 子供 の 頃 に 聴いた 歌 を 1 曲 ずつ 共有

  [ ①  を 今夜 の 約束 に する ]  [ ②  ]  [ ③  ]

─ ふたり の 暦 (直近 7 日) ────────────────────────────────
  05-16  ピザ の 日       ピザ を 半分 ずつ 選ぶ           ✓
  05-15  国際 看護 の 日  自分 を 看護 する 5 分            ✓
  05-14  なし
  05-13  母 の 日 (繰越)  母 へ LINE を 2 人 で 1 通       ✓
  05-12  --------
  05-11  ---------
  05-10  ---------
```

## なぜ FastAPI + ダッシュボード か

- 2 人 の 端末 (スマホ ・ PC) を 同じ LAN で 同じ 画面 を 見る 想定。 サーバー が 1 つ あれば 同期 さ れる
- 認証 / アカウント 不要 — 家 の Wi-Fi に いる 人 が 全員 「ふたり」
- 朝 / 昼 / 夜 で 状態 が 進む (朝 確認 → 夜 やった?)、 ステート は サーバー で 1 つ

## 思想

- 「2 人 で 30 分 ある と し て その 内 5 分」 が 設計 の 基準。 10 分 以上 か かる 提案 は 出さ ない
- 今日 やら なかった も OK、 翌日 「昨日 は なし」 が 並ぶ だけ。 連続 達成 / streak は 出さ ない
- 「今夜 の 約束」 は 1 日 1 つ、 上書き 可能、 やった ボタン で 確定
- 競争 / 評価 数字 / 「ふたり 度」 等 の 採点 は 出さ ない
- 過去 1 年 を 振り返る 「ふたり 年表」 は シンプル な テーブル

## 機能

| 機能 | 内容 |
|---|---|
| 今日 の 暦 | 365 件 の キュレーション 済 micro-holiday、 同日 同じ もの が 出る |
| 3 つ の ritual 提案 | 各 holiday に つき 3 つ の 5 分 ritual を テンプレート で 生成 |
| 今夜 の 約束 | タップ で 1 つ を 確定、 上書き 可能 |
| やった ね | 約束 を 「やった」 に する、 夜 23 時 まで に は 何 もし なくて OK |
| 直近 7 日 / 30 日 暦 | テーブル で 振り返り、 やった / なし が 1 行 ずつ |
| 365 表 | 1 年 通し の カレンダー ビュー (全 完走 後 解放) |

## 起動

```bash
# 依存 を 入れる
pip install -e ".[dev]"

# 開発 サーバー
uvicorn app.main:app --reload --port 8765

# 開く
# 同じ Wi-Fi の 2 台 から: http://<your-ip>:8765/
# 自分 だけ: http://localhost:8765/
```

## テスト

```bash
pytest
```

純ロジック (holidays / rituals / store) を pytest で。 ルーティング も TestClient で smoke。

## データ 保存

`data/state.json` (デフォルト、 `FUTARI_DATA` 環境変数 で 上書き 可):
- `tonight_promise` — 今夜 の 約束 (date, holiday_id, ritual_index, status, note)
- `timeline` — 過去 365 日 の 履歴 (各 日 に 1 件 まで)
- `settings` — ふたり 名 (任意)
