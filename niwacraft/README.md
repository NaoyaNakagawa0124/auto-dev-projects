# niwacraft（庭クラフト）

> 親子で楽しむコージーな2D庭づくりゲーム 🌱🏡

グリッドベースの庭づくりシミュレーション。作物・花・木・小道・柵・水場などを自由に配置して、理想の庭をデザインしよう。季節が巡り、植物が成長し、収穫の喜びを体験できる。

## Features

- **自由な庭づくり** — 16x12グリッドに好きなアイテムを配置
- **季節サイクル** — 春→夏→秋→冬が巡り、植物の見た目が変化
- **植物の成長** — 種まき→芽→成長→収穫の4段階
- **食育ヒント** — 各作物に食育豆知識を表示
- **多彩なアイテム** — 野菜、花、果樹、小道、柵、ベンチ、池など
- **コージーなビジュアル** — やさしい色合いのドット絵風グラフィック

## Tech Stack

- Godot 4.x
- GDScript
- Built-in drawing (no external assets needed)

## How to Run

1. [Godot 4.x](https://godotengine.org/) をインストール
2. Godotでこのプロジェクトフォルダを開く
3. F5 または ▶ ボタンで実行
