# niwacraft

Godot 4.x game project. 2D grid-based garden building simulation.

## Run
Open in Godot 4.x editor and press F5.

## Test
Tests use GUT (Godot Unit Testing) framework.
Run from Godot editor: `res://tests/` directory.

## Conventions
- All user-facing text in Japanese
- GDScript with snake_case
- Scene files in scenes/
- Scripts in scripts/
