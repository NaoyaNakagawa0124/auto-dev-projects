# Wancostar — Implementation Plan

## Phase 1: Swift Project & Data Model
- Swift Package with executable target
- Walk data model (date, duration, distance, mood, route, notes, infrastructure rating)
- Dog profile model (name, breed, birthday)
- JSON file persistence (~/.wancostar/data.json)
- CLI-based app with beautiful terminal output (no Xcode required)

## Phase 2: Walk Management
- Add walk command with interactive prompts
- List walks with formatted table
- Walk statistics (total, average, streaks)
- Dog profile setup

## Phase 3: Cosmic Visualization
- ASCII/Unicode art starfield in terminal
- Each walk = a star character (size/brightness varies)
- Genre clustering by route type
- Monthly constellation groupings
- Beautiful colored terminal output

## Phase 4: Infrastructure Rating & Polish
- Rate sidewalks, parks, shade on walks
- Infrastructure summary statistics
- Japanese text throughout
- Help command and documentation
