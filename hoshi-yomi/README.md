# 星詠み (hoshi-yomi)

> 星座を辿って語彙を覚える、夜空の語学体験。Godot 4 で動く静かな夜の作品。

## これは何？

**星詠み** は、星をなぞって外国語の単語を覚える Godot 4 のゲームです。

各単語に星座が割り当てられ、画面に浮かぶ星を順番にタップ／クリックして繋ぐと、星座の名前が現れます。連なった星は金色に灯り、夜空に永遠に残り続けます。覚えた星座は「夜空辞書」として蓄積されていきます。

語学の暗記が辛い人のための、「美しい夜空を眺めていたら気がつけば覚えていた」という体験を目指しています。

## 特徴

- **手書きの星座**: 12 個の単語に対応する星座を、デザイナーが手作業で配置
- **静かな夜空**: 数百個の小さな星がゆっくり回転する背景。流れ星が稀に流れる
- **金の軌跡**: 正しい順序で星を結ぶたびに、金色の光が灯る
- **シェーダ**: 星のグロー・twinkle はカスタム gdshader で表現
- **夜空辞書**: 解き明かした星座は永久に夜空に残る (UserData に保存)
- **完全日本語UI**: ボタン・案内・単語ラベルは全て日本語＋ターゲット言語表記

## 動かし方

### Godot 4.x で開く

```bash
# Godot 4.2+ をダウンロード: https://godotengine.org/
# 起動 → "Import" → hoshi-yomi/project.godot を選択
# あるいは CLI から:
godot --path hoshi-yomi
```

### データのバリデーション

Godot がなくても、データの整合性チェックは Python で可能です。

```bash
python3 tests/validate.py
```

## 操作

- 左上の「夜空辞書」をクリックして覚えたい単語を選ぶ
- 星座の星が薄く現れる（番号順）
- 順番通りに星をタップ／クリックする
- 全部つなぐと、単語が夜空に浮かび上がる
- ESC で辞書に戻る

## 技術スタック

- Godot 4.2+ (GDScript)
- 手書き .tscn シーンファイル
- カスタムグローシェーダ (gdshader)
- JSON データロード (FileAccess + JSON.parse_string)
- ユーザー進捗保存: user://progress.json
