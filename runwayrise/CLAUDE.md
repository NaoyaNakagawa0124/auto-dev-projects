# RunwayRise

Chrome extension (Manifest V3) with vanilla JS.

## Test
```bash
node tests/test.js
```

## Conventions
- All game logic in js/ directory
- No external dependencies
- Chrome Storage API for persistence
- Spring 2026 color palette: cobalt (#0047AB), cherry red (#DE3163), canary yellow (#FFEF00), violet (#7F00FF)
