# Implementation Plan

## Phase 1: Data Layer
- SQLite database schema (trips, entries)
- CRUD operations for trips and journal entries
- Timestamp and location tracking

## Phase 2: Question Engine
- Psychology-based question bank (nostalgia, sensory, emotional)
- Time-of-day aware question selection
- Trip-progress aware prompts (beginning, middle, end)

## Phase 3: CLI Interface
- Interactive write mode with gentle prompts
- Read mode with beautiful formatting
- Trip management (start, end, list)
- ANSI color theming with warm tones

## Phase 4: Letter Generator
- Compile entries into narrative "letter to future self"
- Emotional tone analysis (keyword-based)
- Beautiful formatted output
- Markdown export

## Phase 5: Polish
- Trip statistics and summary
- Edge cases and error handling
- Tests
