# ChakraForge - CLAUDE.md

## Project Overview
p5.js generative art web app creating anime-style energy effects with narrative skill progression.

## Tech Stack
- Vanilla HTML/CSS/JS + p5.js (CDN)
- No build step, no npm

## Running
Open index.html in a browser, or `npx serve .`

## Testing
Open test.html in a browser for visual tests, or run `node tests/test.js` for unit logic tests.

## Conventions
- Single-file core (sketch.js) with technique modules
- camelCase for variables/functions
- Dark theme: #0a0a0f background
- Color palette per technique defined in techniques object
