# 🍙 おむすび — 遠距離カップル食事共有アプリ

おにぎりマスコット「おむすび」と一緒に、離れていても「一緒にごはん」の体験を。

## コンセプト

遠距離恋愛中のカップルが、食事の記録を共有してつながりを感じるアプリ。マスコットキャラ「おむすび」がリアクションしてくれます。

## 機能

- **食事ログ** — 二人それぞれの食事を記録
- **一緒にごはん判定** — 同じ時間帯に食事すると「一緒にごはん」ストリーク
- **おむすびリアクション** — マスコットが食事内容に応じてコメント
- **レシピ提案** — 二人で同時に作れるレシピを提案
- **統計** — 食事ストリーク、好きな料理ランキング

## 技術スタック

- **Dart** — コアロジック
- **HTML/CSS/JS** — Web UI
- **LocalStorage** — データ永続化

## 使い方

### CLI
```bash
dart pub get
dart run bin/omusubi.dart log "カレーライス"
dart run bin/omusubi.dart status
dart run bin/omusubi.dart suggest
```

### Web
```bash
cd web && python3 -m http.server 8080
```
