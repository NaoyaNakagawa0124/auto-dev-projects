# yasai-nikki — プロジェクトコンベンション

- C99 + Raylib 5.5
- 全 UI 日本語、ひらがな多め (小学校低学年〜高学年向け)
- 漢字はルビなしで使える範囲のみ (小学校 4 年生までを目安)
- カラー: soil #5b3924, leaf-light #7fbd5e, leaf-deep #3a6c2a, sky #cee8ff, cream #fef7d6, sunset #ffb35c, soft-pink #ffcce0, ink #3b2f23
- フォント: macOS の Hiragino Sans を LoadFontEx
- 永続化: `~/.yasai-nikki.json` (HOME 環境変数から)
- 観察データは生徒の作品。データは絶対に破棄しない、上書きしない
- Pixel art は 16x16 を 4 倍拡大 (64x64 で描画) を意識
