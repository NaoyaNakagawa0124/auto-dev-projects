# やさい日記 (yasai-nikki)

> 小学生のための、毎日のやさい観察日記。28 日後に「観察カレンダー」が完成する。

## これは何？

**やさい日記** は、6 〜 12 歳の子どもが毎日 1 分だけ開く、ピクセルアートの野菜観察ゲームです。

- 5 種類の野菜から好きなものを 1 〜 3 つ選んで「植える」
- 1 日 1 回開いて、3 つの観察カードに答える (タップだけ — 入力なし)
- 種から芽、葉、つぼみ、開花、実、収穫まで 28 日かけて育つ
- 28 日後、その日の観察と気分が並んだ「観察カレンダー」が完成する
- 何年後に開いても、その日の自分の発見が残っている

10 年後に「あの夏、ぼくはトマトをこんな目で見ていたんだ」と思える日記を、Raylib のピクセルアートで作りました。

## 特徴

- **5 種類のやさい**: トマト、キュウリ、ピーマン、ナス、トウモロコシ
- **6 段階の成長**: タネ → 芽 → 葉 → つぼみ → 花 → 実 (収穫)
- **3 つの観察カード**: 葉の数 / 色 / きょうのきもち
- **タップ操作のみ**: キーボード入力なし。← → ↑ ↓ Enter だけ
- **観察カレンダー**: 28 日分の観察が一覧表示される、A4 風の最終ページ
- **ファイル保存**: ホームディレクトリの `.yasai-nikki.json` に永続化
- **ピクセルアート**: 矩形と楕円だけで描く、温かい絵

## 動かし方

```bash
brew install raylib   # 前提
cd yasai-nikki
make
./yasai-nikki

make test
```

## 操作

| キー | 操作 |
|---|---|
| ← → | 選択肢を切り替え |
| Enter / Space | 決定 |
| Q | 観察カレンダーを見る |
| R | 新しい日記を始める (確認あり) |
| ESC | 終了 (進捗は自動保存) |
