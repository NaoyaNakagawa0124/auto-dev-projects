# ミーム札 (meme-fuda)

> 二人で 1 枚ずつ、家の思い出をミーム札にする Textual TUI。

孫 (or 家族の誰か) がノートパソコンを持って祖父母の家に行く。 二人で
1 つの画面を見ながら、おばあちゃんが「あの日、お父さんが…」と話す。
孫が、12 種類の **かわいい顔文字テンプレート** のひとつを選び、
上の行に状況、下の行に落ちを書く。── ミーム札ができる。

カードはローカルの "家のデッキ" (`~/.meme-fuda/deck.json`) に積まれる。
日付・話した人・書いた人がタグ付けされる。 数年後、 一緒にデッキを
めくり返して、 また笑える。

> 二人で開けるか? — Intent 7 (誰かと一緒にやる) の判定。
> このアプリは、 起動直後に 「話す人は?」 「書く人は?」 と聞いてくる。
> 一人で起動しても作れるけれど、ふたりで開いた方が、ちゃんと楽しい。

## Why

シニアの孤独 ✕ 孫が遊びに来てもネタが続かない ✕ でも昔話は無尽蔵
— このギャップを「一緒に作る作業」で埋める。 ミームの形を借りるけど、
中身は家の歴史。 出力は SNS に貼るためのものではなく、 内輪で読み返す
ためのもの。

## What

- 12 種類のミーム札テンプレート (顔文字 + 構図)
- 「上のテキスト = 状況」 「下のテキスト = 落ち」 の 2 枠
- 話した人 / 書いた人 / 日付 / タグを記録
- デッキ画面でぱらぱらめくれる
- 「今日の一枚」 起動時にランダムに昔の札を 1 枚見せる

## Tech

- Python 3.10+
- Textual 1.x + Rich
- JSON atomic write at `~/.meme-fuda/deck.json`
- pytest + pytest-asyncio + Textual `run_test` pilot

## Install

```bash
pip install -e .
meme-fuda
```

## Quality goal

- 顔文字は心がほぐれる柔らかい系 (◕‿◕) (´• ω •`) など、 棘のないもの
- カードは和紙のような薄い枠線、 余白多め
- フォントは大きめ (シニアが読めるサイズ)
- 派手な色は使わない。 鈍金・墨・薄紅
