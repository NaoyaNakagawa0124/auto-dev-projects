# Implementation Plan

## Phase 1: Sky Rendering Engine
- Canvas-based sky gradient (time-of-day aware)
- Sun/moon positioning
- Star field for nighttime
- Cloud rendering

## Phase 2: Oshi Color Integration
- Color picker / preset palette
- Oshi name input
- Blend oshi color into sky gradients
- Save to localStorage

## Phase 3: Weather & Season
- OpenMeteo API integration
- Weather-to-visual mapping (rain, clouds, clear, snow)
- Season detection + seasonal particles (blossoms, fireflies, leaves, snow)
- Geolocation for local weather

## Phase 4: Polish
- Ambient messages overlay
- Smooth transitions between states
- Setup modal for first-time users
- Japanese UI throughout
- Responsive fullscreen
