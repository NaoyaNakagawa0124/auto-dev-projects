# 推し神社 (OshiShrine) — あの頃のインターネットで推しを祀ろう

2000年代のGeoCities風ファンサイトを、あなたの推しのために作ろう！

## 特徴

- **レトロデザイン** — 星空背景、キラキラカーソル、虹色マーキーテキスト
- **推しカスタマイズ** — 名前、キャッチフレーズ、テーマカラーを設定
- **ノスタルジック演出** — 訪問者カウンター、Under Construction、ゲストブック風
- **キラキラエフェクト** — Canvas パーティクルで90年代のキラキラを再現

## Tech Stack

- Vanilla JS / Canvas API
- Vite (build)
- 90s CSS aesthetics (no modern frameworks intentionally)

## 実行方法

```bash
npm install
npm run dev
```
