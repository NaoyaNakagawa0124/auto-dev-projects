# 積ん読 (TsunDoku) — 本を積め、棚を埋めろ！

読書家のための、本棚パズルゲーム。落ちてくる本を棚にピッタリ詰め込め！

## 遊び方

1. **棚を設計** — ラウンド開始前に棚のレイアウトを選ぶ
2. **本を積む** — 様々なサイズの本が落ちてくる。左右に動かして棚に配置
3. **行を埋める** — 棚の行を完全に埋めるとスコアゲット＆消える
4. **ラウンド制** — 3ラウンドの合計スコアで勝負！

## 操作

- **← →**: 本を左右に移動
- **↓**: 高速落下
- **Z**: 本を回転
- **Space**: ハードドロップ
- **Enter**: 決定

## Tech Stack

- C / Raylib

## ビルドと実行

```bash
make
./tsundoku
```
