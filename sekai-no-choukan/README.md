# 世界の朝刊 (sekai-no-choukan)

> 就活生 の ための、 5 分 で 読む 12 本 の 世界 ニュース。 Electron 製 の 美しい 朝刊。

朝、 コーヒー を 淹れた 5 分間 だけ、 Mac の デスクトップ で 静か に 開く。 3 段組 で 並ぶ 12 本 の 見出し。 世界 6 地域 × カテゴリ 4 種 (政治 / 経済 / 文化 / 科学) の バランス。 セリフ系 タイポグラフィ、 紙色 の 背景、 余白 多め。 戻る ボタン も SNS シェア も 無い。

明日 の 面接 で 「最近 の 国際 情勢 で 興味 ある の は?」 と 聞かれた 時、 「先週 の 朝刊 で 〇〇 を 読みました」 と 答えられる、 そんな 朝 の 儀式 を 提供 する。

## できること

- 12 本 の 世界 ニュース (2026 年 5 月 中旬 を 想定、 curated + 創作)
- 6 地域 (北米 / 欧州 / 東アジア / 東南アジア / 南米 / アフリカ) を カバー
- 3 段組 グリッド の 自動配置、 1 本 は 大きく 取り上げる 「主見出し」
- 紙色 (#faf6ee) 背景、 セリフ系 フォント、 列罫線、 ドロップキャップ風 の 大見出し
- macOS では vibrancy 効果 で 「窓越し に 透ける」 感じ
- 5 つの 操作 だけ: 開く、 閉じる、 別 日 (前後 7 日 範囲)、 リフレッシュ (同じ シード)、 終了

## 哲学

- 「美しさ で 殴る」 が 魂。 機能 を 増やさない、 紙 1 枚 を 美しく 仕上げる
- 「最新」 を 名乗らない。 公開 情報 ベース の 静的 データ + 創作、 「2026 年 5 月 時点 を 想定」 と 明記
- 5 分 で 閉じられる 朝 の 儀式 — タブ も 通知 も 履歴 も 無し

## 走らせる

```bash
cd sekai-no-choukan
npm install              # electron ~200MB + vitest
npm test                 # ピュア ロジック ユニット テスト
npm start                # Electron で 起動
```
