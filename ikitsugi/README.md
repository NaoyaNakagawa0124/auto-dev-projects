# 息継ぎ (ikitsugi)

> ボス戦中の「息止め」をやめさせる、効率厨ゲーマーのための極小ブラウザ拡張。

## これは何？

**息継ぎ** は、ランクマッチや配信を見ている間に呼吸が浅くなる効率厨ゲーマーのために、画面の隅で静かに呼吸を続けてくれる Chrome / Firefox 拡張機能です。

- どのタブにも、右下にぽつんと光る「呼吸ドット」が現れる
- ドットは設定した呼吸法 (ボックス / 4-7-8 / 4-4-8) のリズムでゆっくり膨らみ・縮む
- ドットを見ているうちに、自然と呼吸がそれに合っていく
- クリックすると 60 秒のガイド付きセッションへ
- フルスクリーン中・ゲームサイト中は自動で透明度を落として邪魔しない
- 1 日に何回息継ぎしたかが小さく記録される

「邪魔しない」「気づかれない」「いつもそこにいる」が設計の根幹です。

## 特徴

- **3 つの呼吸法**: 4-4-4-4 ボックス / 4-7-8 リラックス / 4-4-8 集中
- **どこにいても** : 全タブの右下 (位置は変更可) にひとつの呼吸ドット
- **そっと寄り添う**: ボタンを押さない限り何も鳴らない、振動しない、ポップアップしない
- **ゲーム集中時は自動退避**: フルスクリーン or document hidden で透明度 0
- **ガイドセッション**: クリックで 60 秒間、画面中央に拡大した円が呼吸案内
- **ささやかな自己肯定**: 1 日のセッション数、累計呼吸サイクル数をポップアップに表示
- **設定**: 呼吸法、ドットの大きさ、位置、開始の有効/無効
- **完全日本語UI**

## インストール

### Chrome / Edge

```bash
# 開発者モードで読み込み
1. chrome://extensions/ を開く
2. デベロッパーモードを ON
3. 「パッケージ化されていない拡張機能を読み込む」 → src/ フォルダを指定
```

### Firefox (デバッグ用)

```bash
1. about:debugging#/runtime/this-firefox を開く
2. 「一時的なアドオンを読み込む」 → src/manifest.json を指定
```

node --test "tests/*.test.mjs"

## 操作

| 操作 | 動作 |
|---|---|
| ドットをクリック | 60 秒のガイドセッション開始 |
| セッション中にもう一度クリック | セッション終了 |
| ESC | セッション終了 |
| ツールバーアイコン | 設定 / 統計 ポップアップ |
