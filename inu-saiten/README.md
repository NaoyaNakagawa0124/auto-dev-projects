# 犬採点 (inu-saiten)

> もし あなた の 犬 が レシピ 評論家 だったら、 こう 言う だろう。

レシピ サイト を 開く と、 ページ 上部 に 「あなた の 犬 の 評価」 が 自動 で 載る Chrome 拡張。 材料 を 解析 して、 犬 の 主観 100% で 採点 + 1 段落 の レビュー が 出る。 「肉 多めで ナイス。 玉ねぎ さえ なければ 5 つ星 だった。 −1 点。 でも、 香り だけ で 私 は もう 幸せ です。」

```
┌──────────────────────────────────────────────────────────────┐
│  🐕  あなた の 犬 の 評価                       ★★★★☆ 88/100  │
├──────────────────────────────────────────────────────────────┤
│  > 肉 が 主役 で 大 賛同。 鶏胸肉 の 香り だけ で 1 日 過ごせる。  │
│  > 玉ねぎ が 入って いる の が 痛い、 私 は 部屋 の 隅 に 退避     │
│  > した。 でも 食べ終わった お皿 の 「お こぼれ」 を 期待 して、  │
│  > 私 は 結局 キッチン に 戻り ます。 — お利口 ポチ              │
│                                                              │
│   🐕 命 を 賭ける         鶏胸肉 (95)  ささみ (90)  鶏ガラ (85) │
│   ⚠️  部屋 の 隅 に 退避   玉ねぎ (-)   ニンニク (-)  チョコ (-) │
│   😴  正直 どう でも       薄力粉 (10)  砂糖 (20)   塩 (5)     │
└──────────────────────────────────────────────────────────────┘
```

## なぜ Chrome 拡張 か

- 既存 の レシピ サイト (cookpad / クラシル / DELISH KITCHEN / NYT Cooking / AllRecipes) を そのまま 使い ながら、 「犬 視点」 だけ を 重ねる の が ポイント。 アプリ を 開く 必要 が ない、 普通 に レシピ を 見て いる だけ で 犬 が 採点 して くれる
- popup で 直近 「採点 した レシピ」 履歴 を 見返せる、 飼い主 の 犬 偏重 観 が 静か に 育つ

## 思想

- これ は **コメディ拡張** で あって 栄養 指導 で は ない。 「玉ねぎ は 犬 に 危険」 等 の 一般 知識 は ベース に する が、 採点 と レビュー 文 は 全部 笑い 目的
- 犬 の キャラ は 4 種 (お利口 ポチ / シニア の マロ / ヤンチャ な タロウ / 哲学者 ジョン)、 同じ レシピ でも キャラ で 評価 が 変わる
- 投稿 / 共有 機能 は なし — ローカル の 自分 と 犬 だけ で 楽しむ

## 機能

| 機能 | 内容 |
|---|---|
| 自動 採点 オーバーレイ | レシピ ページ で 材料 を 検出 し ページ 上部 に 評価 カード を 注入 |
| 4 つ の 犬 キャラ | お利口 ポチ / シニア マロ / ヤンチャ タロウ / 哲学者 ジョン (popup で 切替) |
| 危険 材料 警告 | 玉ねぎ / ニンニク / チョコ / 葡萄 / アボカド 等 は 「⚠️」 マーク (情報 表示 のみ、 飼育 助言 ではない 旨 を 注釈) |
| 履歴 popup | 直近 20 件 の レシピ URL + 採点 を 履歴 表示 |
| 「ご褒美 リスト」 | 採点 90+ の 材料 を 「犬 が 命 を 賭ける リスト」 に 集計 |
| 言語 切替 | UI は 日本語 固定 (材料 検出 は 日本語 + 英語) |

## 動かす

1. Chrome で `chrome://extensions/` を 開く
2. 右上 「デベロッパー モード」 を ON
3. 「パッケージ 化 されて いない 拡張 機能 を 読み込む」 で `inu-saiten/extension/` フォルダ を 選択
4. レシピ ページ を 開く と 上部 に カード が 出る

## テスト

```bash
npm install
npm test
```

純ロジック (材料 検出 / 採点 / レビュー 生成) は Vitest で。 DOM 操作 は smoke のみ。

## データ 保存

- `chrome.storage.local`:
  - `inu-saiten:settings` — 選択 中 の 犬 キャラ、 表示 ON/OFF
  - `inu-saiten:history` — 直近 20 件 の レシピ URL + 採点
