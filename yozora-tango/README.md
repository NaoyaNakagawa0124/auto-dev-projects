# 夜空単語 (yozora-tango)

> 深夜 2 時の単語暗記を、 5 分の星座コレクションに変える。

英単語カードが星として夜空に流れてくる。 「思い出せた」 を 1 タップすると星が点灯し、 隣の星と線で結ばれていく。 4-7 個の星をつなぐと 1 つの星座が完成し、 「夜空コレクション」 に保存される。 1 セッションは 5 分。

詰め込み暗記の単調さを、 「星座を完成させたい」 という小さな中毒に置き換えるための実験。 streak も連勝も無いが、 集めた星座は 10 年経っても残る。

## 走らせる

```bash
cd yozora-tango
npm install
npm test               # 30+ ユニットテスト

# 配信 — プロジェクトルートから起動
python3 -m http.server 8080
# http://localhost:8080/web/
```

p5.js は CDN から読み込む (オフラインでも動くようにしたい場合は `web/lib/` に `p5.min.js` を置く)。

## 5 つの星座

| 名前 | 構成点数 |
|------|---------|
| オリオン | 7 |
| はくちょう | 5 |
| カシオペア | 5 |
| こぐま | 7 |
| ペガサスの大四辺形 | 4 |

合計 28 個の星を、 30 個の暗記カードから埋めていく構造。
