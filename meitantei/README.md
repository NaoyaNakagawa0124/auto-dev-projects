# 名探偵 (MeiTantei) — 写真から真実を暴け

就活中のあなた。ある日、不可解な事件の証拠写真を手に入れた。写真を調べ、手がかりを見つけ、真相を解明せよ。

## 特徴

- **証拠写真の調査** — 写真をクリック・ズームして隠された手がかりを発見
- **推理ボード** — 見つけた手がかりをボードに並べて繋げる
- **3つの事件** — 難易度の異なる3つのミステリーを解決
- **分岐する推理** — 間違った推理をすると別の結末に
- **ダーク＆ミステリアス** — サスペンス感あふれるUI

## Tech Stack

- Vanilla JavaScript / HTML5 / CSS3
- Vite (build)
- Canvas API for photo investigation
- localStorage for save progress

## 実行方法

```bash
npm install
npm run dev
```
