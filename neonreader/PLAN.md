# Neon Reader — Implementation Plan

## Phase 1: Extension Foundation & Reader Mode
- Manifest V3 setup (content script, popup, background)
- Content script that extracts article text from any webpage
- Reader mode overlay that replaces page content
- Cyberpunk dark theme with neon text

## Phase 2: Accessibility & Customization
- Font size slider (large default for seniors)
- Color theme options (neon green, neon cyan, neon amber)
- Line height and letter spacing controls
- Reading guide (highlighted line tracker that follows scroll)

## Phase 3: Cyberpunk HUD & Effects
- HUD overlay showing word count, reading time, scroll progress
- Subtle scanline CSS effect
- Text reveal animation (typewriter effect on first load)
- Neon glow effects on headings and UI elements

## Phase 4: Polish & Tests
- Japanese UI throughout
- Settings persistence in chrome.storage
- Keyboard shortcuts
- Test suite
