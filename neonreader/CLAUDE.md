# Neon Reader Project Conventions

- Chrome Extension Manifest V3
- Vanilla JS + CSS (no framework)
- All user-facing text in Japanese
- Cyberpunk color palette: dark bg (#0a0a0f), neon green (#00ff41), neon cyan (#00f0ff), neon amber (#ffb000)
- Large, readable fonts — minimum 18px body text
- Accessibility first, cool aesthetics second
