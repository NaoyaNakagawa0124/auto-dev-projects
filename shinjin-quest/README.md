# 新人クエスト — 新卒サバイバルシミュレーション

入社1年目のリアルを体験するターミナルサバイバルゲーム。4つのリソースを管理しながら365日を生き残れ！

## ゲーム概要

あなたは新卒社員。毎日の選択が体力・メンタル・貯金・評価に影響する。
残業するか？飲み会に行くか？有給を使うか？
1年間を乗り越え、最高の評価を目指せ。

## リソース

- **体力 (HP)** — 0になると入院 → ゲームオーバー
- **メンタル (MP)** — 0になると休職 → ゲームオーバー
- **貯金 (¥)** — マイナスになると生活破綻 → ゲームオーバー
- **評価 (★)** — 高いほどボーナスUP、低すぎると解雇

## 特徴

- **365日のリアルな日常** — 平日は仕事、週末は自由行動
- **ランダムイベント** — 急な残業、飲み会のお誘い、台風で電車遅延...
- **季節の変化** — GW、お盆、年末年始などリアルなカレンダー
- **複数エンディング** — 評価に応じて5種類のエンディング
- **スコアシステム** — 総合スコアで実力を競え

## ビルド & 実行

```bash
cd shinjin-quest
make
./shinjin-quest
```

## 技術スタック

- **C++17** — ゲームロジック
- **ANSI Terminal** — カラー出力・UI描画
- **Makefile** — ビルドシステム
