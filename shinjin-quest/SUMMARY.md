# 新人クエスト — Summary

## What Was Built

A C++ terminal survival simulation game where you play as a new graduate navigating their first year at a Japanese company. Manage 4 resources (体力, メンタル, 貯金, 評価) through daily choices over 365 days. Features seasonal events (GW, bonuses, New Year), 15+ random events, and 5 different endings.

## Features

- **365-Day Calendar** — Real Japanese calendar with holidays, weekends, seasons
- **4 Resource System** — HP (体力), MP (メンタル), Money (貯金), Rating (評価)
- **15+ Random Events** — Drinking parties, surprise presentations, train delays, phone disasters
- **Seasonal Events** — GW, summer/winter bonuses, Obon, New Year
- **5 Endings** — 解雇, 燃え尽き, 金欠, 生存, 昇進, 伝説の新人
- **Score System** — Composite score based on final stats
- **ANSI Terminal UI** — Colored bars, box drawing, emoji integration

## Tech Stack

- C++17
- ANSI escape codes for terminal rendering
- Makefile build system
- No external dependencies
