# 真夜中弁当 (MidnightBento) — 深夜の弁当パズル

深夜2時。勉強の合間に、完璧な弁当を詰めろ！

## 遊び方

- 食材を弁当箱にぴったり詰め込むパズルゲーム
- 食材はそれぞれ異なる形（1x1〜3x2）
- 全ての食材を弁当箱に収めればクリア！
- 10ステージ、だんだん難しくなる

## 食材と効果

- 🍙 おにぎり — 集中力アップ
- 🥚 卵焼き — 記憶力アップ
- 🍗 唐揚げ — スタミナ回復
- 🥦 ブロッコリー — 健康ブースト
- 🍣 寿司 — ご褒美ポイント2倍
- 🥟 餃子 — コンボボーナス

## Tech Stack

- Godot 4 / GDScript

## 実行方法

```bash
godot project.godot
```

## 操作

- **クリック**: 食材を選択
- **ドラッグ**: 食材を移動
- **右クリック / R**: 食材を回転
