# 空間 (Kuukan) — Summary

## What Was Built

A space-station-themed virtual office web app for lonely remote workers. Features an animated 3-layer starfield background, mission-based pomodoro timer, daily "transmissions from mission control" with motivational messages, virtual crew members working alongside you, and a mission log. The space station metaphor addresses remote work isolation through immersive theming.

## Features

- **3-Layer Starfield** — Canvas-rendered parallax star animation with twinkling
- **Mission Timer** — Pomodoro-style with 25/50/15-minute presets, mission status badges
- **Mission Control Transmissions** — 15 unique messages with typing effect, practical advice
- **Virtual Crew** — 6 named crew members with randomized tasks and status indicators
- **Mission Log** — Timestamped activity log persisted in localStorage
- **Daily Stats** — Mission count and focus time tracking with daily reset
- **Space Station UI** — Glassmorphism panels, pulse animations, JetBrains Mono typography
- **Responsive** — Works on mobile with single-column layout

## Tech Stack

- Vanilla HTML/CSS/JS
- Canvas 2D (starfield rendering)
- LocalStorage (persistence)
- Noto Sans JP + JetBrains Mono
