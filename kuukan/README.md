# 空間 (Kuukan) — 宇宙ステーション・バーチャルオフィス

リモートワーカーの孤独を、宇宙ステーション体験で癒す。

## コンセプト

あなたのデスクは宇宙ステーションの中。星空を眺めながら、管制室からの通信を受け、バーチャルクルーと一緒に「ミッション」をこなす。リモートワークの孤独感を、没入型の宇宙テーマで解消します。

## 機能

- **🚀 ミッションタイマー** — 25分/50分/15分のポモドーロ式。「発進」で開始
- **📡 管制室通信** — 毎回異なるモチベーションメッセージと実用アドバイス
- **👥 バーチャルクルー** — 一緒に働く仲間の存在感（アクティブステータス表示）
- **📋 ミッションログ** — 作業記録を自動タイムスタンプで保存
- **✨ 星空アニメーション** — 3層パララックスの美しい星空

## 起動

```bash
cd kuukan
python3 -m http.server 8080
# ブラウザで http://localhost:8080 を開く
```

## テスト

```bash
node tests/run.js
```
