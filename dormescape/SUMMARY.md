# DormEscape - Summary

## What Was Built

A Rust+WASM interactive fiction game where a student at 2am navigates their first apartment search. 7 chapters, each teaching a real financial concept (rent ratios, leases, credit, budgeting, emergency funds) through binary story choices. HUD tracks savings, rent, and credit score. Ends with a financial grade based on choices. All data from 2026 housing market research.

## Potential Next Steps

- Multiple storylines based on different cities/salaries
- Shareable results card for social media
- Real apartment listing API integration
- Branching paths with more choices per chapter
