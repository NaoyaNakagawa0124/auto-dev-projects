# DuetPlan - Summary

## What Was Built

A Rust+WASM collaborative web app where two content creators independently swipe on 30 trending topics, then reveal their matches. Built with Rust compiled to WebAssembly for all logic (voting, matching, scoring, plan generation), with a minimal HTML/CSS/JS frontend.

## Potential Next Steps

- Real-time WebSocket sync instead of URL sharing
- Custom topic submission
- Match history across sessions
- Integration with TikTok/YouTube trending APIs
