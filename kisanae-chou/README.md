# 季重ね帖 (kisanae-chou)

> 1 年 分 の アニメ 視聴 履歴 を、 日本 の 季節 (旬 の 野菜 / 二十四 節気 / 桜 前線) に そっと 重ねる 美 ノート。

「あなた が 2026 年 春 に 観た 7 本 の アニメ」 を、 同じ 期間 に 旬 を 迎えた 野菜 や 開花 した 花、 二十四 節気 の 移ろい と 並べて 描く Jupyter ノート。 アニメ オタク が 1 年 を 振り返る とき、 タイトル の リスト だけ で なく、 「あの 4 月 は タケノコ と 桜 と 〇〇 が 一緒 だった」 と 季節 の 文脈 で 思い出 を 厚く 残せる ように。

```
─ 2026 年 季重ね帖 ─────────────────────────────────────

      1月    2月    3月    4月    5月    6月    7月    8月    9月   10月   11月   12月
   ┌────────────────────────────────────────────────────────────────────────────┐
   │  ●    ●    ●    ●●●  ●●   ●●●  ●●●●  ●●●  ●●   ●●●  ●●   ●●     ← アニメ
   │       小寒          春分    立夏    夏至    立秋    秋分    立冬    冬至     ← 節気
   │  ━━ 大根 ━━━┛                            ┝━━━━━━━━ さんま ━━━━┥        ← 旬
   │              桜 開花 ━━━━━┛  あじさい ━━┛                                  ← 花
   └────────────────────────────────────────────────────────────────────────────┘

   出力 4 種:
     ▣ year_timeline.png   — 12 月 横並び の 重ね 表
     ▣ month_04.png        — 4 月 だけ の 拡大 カード (12 枚 出力)
     ▣ season_scroll.png   — 春夏秋冬 縦 巻物 (4 枚 結合)
     ▣ palette_year.png    — 月 ご と の 色 マップ
```

## なぜ Jupyter + matplotlib か

- 「美しさ で 殴る」 系 の 作品 で あって、 アプリ で は な い。 ノート ブック を 開いて 1 回 走らせれば PNG が 揃う
- matplotlib + numpy + PIL で 完全 ロー カル、 依存 軽い
- ユーザー が 自分 の 視聴 履歴 を 差し替える 1 ファイル (`my_anime.json`) を 用意 すれば、 出力 だけ が 変わる

## 機能

- **12 月 横並び タイム ライン** — アニメ の 放映 期間 を 月 別 に 高さ で 表す、 旬 と 節気 を 帯 で 重ねる
- **月 別 カード** (12 枚) — その 月 だけ を 拡大、 アニメ 名 を 縦書き 風 に 配置
- **春夏秋冬 巻物** — 縦 長 PNG (4 枚 連結)、 1 季 = 1 段
- **月 ご と の パレット** — 旬 の 野菜 と アニメ ジャンル の 色 を 混ぜた 12 色 帯
- **サンプル データ** — 2026 年 想定 で 40 本 の アニメ + 旬 食材 30 種 + 二十四 節気 + 桜 / あじさい / 紅葉 開花 日 を 同梱

## 動かす

```bash
# 依存 を 入れる
pip install -e ".[dev]"

# テスト
pytest

# 全 PNG を 生成 (out/ に 出力)
python -m kisanae_chou.run

# Jupyter で 開く
jupyter notebook notebook.ipynb
```

## カスタマイズ

`kisanae_chou/data.py` の `SAMPLE_ANIME` を 自分 の 視聴 履歴 で 上書き する だけ。 各 エントリ は:

```python
{
  "title": "夜 桜 食堂",
  "start": "2026-04-08",
  "end":   "2026-06-24",
  "episodes": 12,
  "genre": "drama",   # action / drama / slice_of_life / mystery / fantasy
}
```

## デザイン 思想

- 「美しさ で 殴る」 — 強い 色 を 使わ ず、 和 紙 風 の 淡い グラデーション
- フォント は matplotlib デフォルト + 日本語 (Hiragino) フォール バック
- 余白 を たっぷり 取り、 タイト ル の 詰め込み は し ない (せいぜい 月 5 本)
- 季節 の 帯 が 主役、 アニメ は 「重なる 印」 と して の 小 円 で 表す
- 出力 は SNS で 共有 し たく なる 縦長 / 横長 サイズ (1200×1500 等)
