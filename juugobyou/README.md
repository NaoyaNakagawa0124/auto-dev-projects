# 十五秒 (juugobyou)

> 部屋を片付けられない自分を責めてしまう人のための、 15 秒だけの小さなボタン。
> Rust + WASM の極小 PWA-風 Web アプリ。

「片付けが苦手」 と検索すると、 出てくるのは 「習慣化のコツ 10 選」 「これで
変わった!」 みたいな、 明るくて 元気な記事ばかり。 ── でも、 ほんとうに
ぐったりして床の山を見つめている人にとって、 そういう言葉はもう 1 つの石。

十五秒は、 そういう人のためにある。

## 1 画面、 1 ボタン

```
              ▒▒▒▒▒▒▒▒▒▒
            ▒▒          ▒▒
          ▒▒    今だけ    ▒▒
            ▒▒          ▒▒
              ▒▒▒▒▒▒▒▒▒▒

           今日、 3 回さわった
```

ボタンを押すと、 15 秒のカウントダウン円が現れる。
中央に、 おだやかな声で、 ひとつの小さな促し:

> 「視界にある、 何かを 1 つだけ、 元の場所に置こう。」

15 秒経つと、 画面はそっと薄くなって、

> 「ありがとう。 ここまでで十分です。」

それだけ。 スコアも、 連続日数も、 通知もない。

## 声を選べる

「うるさい言葉」 が一番のスイッチオフ。 だから声を 3 つ用意した:

- **静か** ── デフォルト。 短い、 主語のない言葉。
- **友達** ── 「ちょっとだけ、 一緒にやろう?」 のトーン。
- **おかあさん** ── 「今日もよく起きたねぇ」 から始まる声。

「今この声、 きついな」 と感じたらすぐ切り替えできる。

## なぜ Rust + WASM

- オフライン完結。 部屋でぐったりしている時に、 ネットが届かなくても動く
- 起動が一瞬。 「やる気が出てから 0.5 秒以内」 に画面が立ち上がる
- バイナリは 70 KB 以下。 古いスマホでも軽い

## 動かす

```bash
wasm-pack build --target web --release --out-dir web/pkg
python3 -m http.server -d web 8081
# http://localhost:8081
```

テスト:
```bash
cargo test
```

## 守ること (CLAUDE.md にも書いた)

- 「頑張れ」 「怠け」 「汚い」 を絶対に使わない (test で禁止)
- 「達成」 「完了」 「クリア」 系の言葉も避ける
- ボタンの 1 タップが 「触る」 であり、 「やる」 ではない
- カウンタは表示するが、 ストリーク化しない (連続日数の概念を持たない)
