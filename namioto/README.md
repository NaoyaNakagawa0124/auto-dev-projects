# 波音 (Namioto) - 呼吸で波を操る瞑想ゲーム

A one-hand ocean meditation game built with Raylib. Hold spacebar to breathe in (waves rise), release to breathe out (waves fall). Steady breathing creates calm seas and clears the sky. A stress-relief experience for busy couples.

## Controls

- **スペースキーのみ** — 押して吸う、離して吐く
- 安定したリズムで波が穏やかになり、空が晴れる
- 乱れたリズムは嵐を呼ぶ

## Features

- 手続き的に生成される美しい海と空
- 呼吸リズムに連動する波の物理
- 穏やかさスコア（ふたりの合計記録）
- 時間帯で変わるカラーパレット
- 片手だけで操作可能

## Build & Run

```bash
make && ./namioto
```
