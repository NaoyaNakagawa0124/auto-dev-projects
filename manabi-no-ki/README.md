# 学びの木 — 子どもの学習で育つ木

学習記録で仮想の木が成長する、親子のためのターミナル学習トラッカー。

## コンセプト

子どもが勉強するたびに、ターミナルの中の木が少しずつ成長する。科目ごとに枝が伸び、学習セッションで葉がつき、マイルストーン達成で花が咲く。

## 機能

- **🌳 成長する木** — ASCII アートの木が学習量に応じて成長
- **📚 科目管理** — 国語、算数、理科など科目ごとの記録
- **⏱️ 学習記録** — 日々の学習時間を簡単に記録
- **🌸 マイルストーン** — 累計時間で花が咲く（10h, 50h, 100h...）
- **📊 統計** — 週間・月間のグラフ表示
- **🏆 ランキング** — 科目別の学習時間ランキング

## 起動

```bash
cd manabi-no-ki
pip install textual  # 未インストールの場合
python3 src/app.py
```

## CLI モード

```bash
python3 src/cli.py log 算数 30     # 算数を30分記録
python3 src/cli.py status          # 木の状態表示
python3 src/cli.py stats           # 統計表示
```

## テスト

```bash
python3 -m pytest tests/ -v
# または
python3 tests/test_all.py
```
