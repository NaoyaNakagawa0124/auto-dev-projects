# ChromaDrip

> It's 2035. Your clothes change color with your mood. This generates the palettes.

## What is ChromaDrip?

In the year 2035, all fashion is programmable. Smart fabrics glow, shift, and pulse with bioluminescent dyes. ChromaDrip is the palette generator every night owl needs — input your mood, the time of day, and the season, and get a 5-color palette optimized for your 2035 glow-reactive wardrobe.

## Features

1. **Mood Palettes** — 12 moods mapped to scientifically-derived color harmonies
2. **Circadian Adaptation** — Palettes shift based on time (night owls get special bioluminescent tones)
3. **Seasonal Drift** — Colors evolve with the season (spring = pastels, winter = deep aurora)
4. **Fabric Simulation** — Shows how colors behave on different 2035 fabric types (neon mesh, liquid crystal, bio-silk)
5. **Palette Export** — Copy hex codes, save as JSON

## Run

bun run src/server.ts
