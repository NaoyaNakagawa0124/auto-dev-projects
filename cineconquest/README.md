# CineConquest - 映画で世界を征服せよ！

A competitive 3D globe visualization where movie lovers "conquer" countries by logging films from around the world.

## Features

- **3Dインタラクティブ地球儀** - Three.jsで描画される美しい地球。映画を観た国が光る
- **映画ログ** - タイトル、国、ジャンル、評価を記録
- **征服システム** - 各国の映画を観るほどその国が明るく輝く
- **ランキング** - 征服率、多様性スコア、連続記録でランク付け
- **ニュースチャレンジ** - 世界のニュースに関連した国の映画を観るとボーナスポイント
- **統計ダッシュボード** - ジャンル分布、国別グラフ、視聴トレンド

## Tech Stack

- Three.js (3D globe rendering)
- Vanilla JavaScript (no framework)
- HTML5 / CSS3 (glassmorphism UI)
- localStorage (data persistence)
- Vite (build tool)

## How to Run

```bash
npm install
npm run dev
```
