# KidStats 2035

A Jupyter notebook set in 2035 showing a futuristic youth multi-sport analytics dashboard. Beautiful Plotly visualizations a busy parent can scan in 5 minutes.

## Features

- Performance trajectory across 8 sports (2032-2035)
- Skill radar with 6 athletic dimensions
- Injury risk heatmap by body zone
- Growth trajectory (actual + AI-predicted to 2038)
- AI-optimized weekly training schedule
- AI Coach recommendation with development analysis
- Customizable: change name, age, seed for different data

## Tech Stack

Python / Jupyter / Plotly / pandas / dataclasses

## Run

```bash
pip install plotly pandas jupyter
jupyter notebook kidstats2035.ipynb
```
