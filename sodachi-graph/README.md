# 育ちグラフ — sodachi-graph

> 親と子の選択が、未来を描く。
> ビジュアルノベル × データダッシュボードで「子育て」を体験する Raylib 製のミニゲーム。

15年分の子育てを5章に圧縮し、各章で2つの選択を迫られます。
選んだ瞬間に「智・徳・体・情」の4スタッツが動き、章末に**成長レポート**で可視化。
最後に到達するエンディングは選択の総和で12種類に分岐します。

## 特長

- **Raylib (C, ネイティブ)** — 1280x720 フルスクリーンレベルの画質、60fps、軽量バイナリ
- **ビジュアルノベル形式** — 5章 × (導入 + 2選択 + 振り返り) = 各章8シーン
- **データダッシュボード** — 章末ごとに4スタッツのバーが滑らかに伸び、デルタ値も表示
- **マルチエンディング** — 12通り（単一強み4 + 二刀流6 + 全人格1 + 凡庸1）
- **完全日本語** — 物語、UI、エンディング、すべて日本語
- **Noto Sans JP 同梱** — 約500種類の日本語コードポイントを動的にロード

## 必要環境

- macOS (テスト済み: Apple Silicon, raylib 5.5)
- `brew install raylib`
- C コンパイラ (clang or gcc)

## ビルド & 起動

```bash
make
./sodachi-graph
```

ウィンドウが開いたら、画面をクリックして物語を進めます（Space / Enter でも進めます）。

## ゲーム内容（ネタバレ少なめ）

5つの章を、それぞれ違う年齢の子どもと一緒に過ごします。

| 章 | 年齢 | テーマ |
|---|------|--------|
| 一 | 0〜2歳 | いのちのはじまり |
| 二 | 3〜5歳 | はじめての社会 |
| 三 | 6〜8歳 | 学校という冒険 |
| 四 | 9〜11歳 | 自分という発見 |
| 五 | 12〜15歳 | 旅立ちの予感 |

各章2回、A/Bの分岐があります。一方は「知性・徳」寄り、一方は「体・情」寄りに振れることが多く、
偏らせれば偏らせるほど鋭いエンディングに、バランスを取れば「全人格的な大人」に到達します。

## スタッツ

- **智** (Chi) — 知性。論理、学習、情報処理。
- **徳** (Toku) — 思いやり。倫理、礼節、社会性。
- **体** (Tai) — 健やかさ。身体、行動、たくましさ。
- **情** (Jou) — 感性。共感、表現、感受性。

すべて50からスタート、上限100。

## エンディング一覧

スタッツの組み合わせで自動選択されます（プレイヤーは知らない方が楽しい部分なので、概要だけ）：

1. 学究の徒 — 智が突き抜けた場合
2. 支える人 — 徳が突き抜けた場合
3. 躍動する人 — 体が突き抜けた場合
4. 感性の人 — 情が突き抜けた場合
5〜10. 二刀流系（賢者、文武両道、知と感性のクリエイター、行動するリーダー、寄り添う支え手、全身で表現する人）
11. 全人格的な大人 — 全スタッツが60以上で差が小さい場合
12. 自分の道を歩む人 — どれにも当てはまらない場合

## 操作

- **左クリック / Space / Enter** — テキストを進める、ダッシュボード次へ
- **A / B ボタンクリック** — 選択

## 開発メモ

このアプリは `/auto-dev` の Discovery Roll で生成されました：

- **Source 40**: データ分析・ダッシュボード系
- **Persona 39**: 子育て中のパパ/ママ
- **Platform 20**: Raylib / SDL game (C/C++/Zig)
- **Wildcard 43**: 選択肢で分岐する（マルチエンディング、ビジュアルノベル的）

「ダッシュボード」と「ビジュアルノベル」は本来異なるジャンルですが、
**章間ダッシュボード**として組み合わせることで、選択の重みを「数値の動き」で実感できる構造に仕上げました。
