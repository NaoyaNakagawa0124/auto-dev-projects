# 学ERP — 勉強エンタープライズリソース管理

あなたの勉強を「企業経営」として管理する、ちょっとユニークなC#コンソールアプリ。

## コンセプト

| 勉強の世界 | 学ERPの世界 |
|-----------|-----------|
| 科目 | 部署 (Department) |
| テスト/試験 | クライアント案件 (Project) |
| 勉強時間 | 請求可能時間 (Billable Hours) |
| 知識 | 在庫 (Inventory) |
| あなたの脳 | 従業員 (Employee) |
| 深夜勉強 | 残業 (Overtime) |
| 休憩 | 有給休暇 (PTO) |

## 機能

- **ダッシュボード** — 全部署の業績サマリー、KPI、週次レポート
- **部署管理** — 科目の追加・進捗確認・優先度設定
- **タイムトラッキング** — 勉強時間を請求可能時間として記録（深夜残業1.5倍）
- **案件管理** — テスト/試験を案件として登録、納期管理
- **在庫管理** — 知識レベルをストック化、減衰あり
- **従業員管理** — 脳の疲労度・集中力モニタリング
- **財務レポート** — 勉強コスパ分析、投資対効果

## ビルド & 実行

```bash
cd gaku-erp
make
mono gaku-erp.exe
```

## テスト

```bash
make test
mono test-runner.exe
```
