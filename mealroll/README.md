# MealRoll

> Spin to discover your next meal. A new-tab recipe roulette with exactly 3 features.

## What is MealRoll?

MealRoll is a Chrome extension that replaces your new-tab page with a beautiful recipe roulette. Every time you open a tab, you're one spin away from discovering what to cook tonight.

**Only 3 features. By design.**

## Features

1. **Spin** — Animated roulette wheel with 40+ recipes. Spin to discover a random meal with ingredients and cook time.
2. **Filter** — Toggle dietary preferences: Vegetarian, Vegan, Quick (<30min), Comfort Food.
3. **Favorites** — Heart a recipe to save it. View your favorites list anytime.

That's it. No accounts. No AI. No meal planning. No grocery lists. Just spin, filter, favorite.

## Tech Stack

- Chrome Extension (Manifest V3)
- Vanilla HTML/CSS/JS
- Chrome Storage API
- CSS animations for the spin

## Install

```bash
# 1. Open chrome://extensions
# 2. Enable Developer Mode
# 3. Click "Load unpacked" → select mealroll/
```
