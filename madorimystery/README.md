# 間取りミステリー (MadoriMystery) — 間取りから事件が生まれる

間取り図を描くと、そこにミステリーが生まれる。謎解き動画のネタに最適！

## 特徴

- **間取り設計** — クリックで部屋を配置。リビング、寝室、キッチンなど
- **ミステリー自動生成** — 間取りに合わせて事件シナリオを自動生成
- **手がかり配置** — 各部屋に手がかりが自動的に散りばめられる
- **捜査モード** — 部屋を調べて犯人を推理
- **シェア機能** — 完成した間取りミステリーをスクリーンショットで共有

## Tech Stack

- Vanilla JS / Canvas API
- Vite (build)
- 間取り描画 + ミステリー生成エンジン

## 実行方法

```bash
npm install
npm run dev
```
