# 映画スケッチ (EigaSketch) — スケッチから映画を当てろ！

手書き風のヒントスケッチを見て、映画のタイトルを当てるクイズゲーム。

## 遊び方

1. ヒントスケッチが徐々に描かれる
2. 4つの選択肢から正しい映画タイトルを選ぶ
3. 早く答えるほど高得点！
4. 30問の映画クイズに挑戦

## 特徴

- **手書き風ビジュアル** — スケッチ風に描かれるヒント
- **6ジャンル** — アクション、SF、ホラー、ロマンス、アニメ、ファンタジー
- **30本の映画** — 有名作品から隠れた名作まで
- **スピードボーナス** — 素早い回答で高スコア

## Tech Stack

- Godot 4 / GDScript
- Procedural sketch-style drawing

## 実行方法

```bash
godot project.godot
```
