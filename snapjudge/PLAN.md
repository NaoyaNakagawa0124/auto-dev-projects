# SnapJudge Implementation Plan

## Phase 1: Core Engine
- Project setup (project.godot, main scene)
- Photo card scene with procedural generation
- Quality attribute system (sharpness, exposure, composition)
- Swipe/keyboard input handling

## Phase 2: Game Loop
- 60-second timer
- Photo queue and card dealing
- Scoring system (speed bonus, accuracy, streak multiplier)
- Sound effects (correct/wrong/combo)

## Phase 3: Results & Polish
- Results screen with statistics
- High score persistence
- Japanese UI throughout
- Visual polish (animations, particles, screen shake)
