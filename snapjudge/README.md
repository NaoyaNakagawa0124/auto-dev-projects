# SnapJudge (スナップジャッジ) - 写真を瞬時に仕分けろ！

写真の「目利き力」を鍛える、スピード写真選別ゲーム。

## 特徴（3つだけ）

1. **判定** — 写真が次々と現れる。素早く「残す」「削除」「スター」を判定せよ
2. **スコア** — スピードと正確性でスコアが決まる。連続正解でコンボボーナス
3. **結果** — ラウンド終了後、あなたの判定力を統計で可視化

## 遊び方

- **→ キー / 右スワイプ**: 残す（良い写真）
- **← キー / 左スワイプ**: 削除（ダメな写真）
- **↑ キー / 上スワイプ**: スター（最高の写真）
- 60秒以内にできるだけ多くの写真を正確に仕分けよう！

## Tech Stack

- Godot 4 (GDScript)
- Procedural photo generation (no external assets)

## How to Run

```bash
# Godot 4をインストールして開く
godot project.godot
```
