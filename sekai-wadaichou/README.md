# 世界話題帳 (sekai-wadaichou)

> 就活中の大学生の深夜 2 時、 ESを書く手が止まったら、 地球を回す。

矢印キーで地球儀を回すと、 12 都市が順番にスポットに来る。
各都市には今日の文化イベントと、 明日の面接でそのまま使える 1 行の小ネタが書いてある。
気に入ったら 1 タップで「話題帳」 に綴じる。

これは:
- 就活疲れの「もうやる気でない」 を、 5 分の世界一周に置き換えるための TUI
- 「面接の雑学帳」 + 「世界の今」 を 1 つにしたもの
- 起動するたびに違う都市から始まる、 1 周しても飽きない設計

## キーバインド

| キー | 動作 |
|-----|------|
| ← / → | 地球儀を回転 (3 都市分の角度) |
| Enter | 現在の都市カードを話題帳に綴じる |
| Tab | 地球儀 ↔ 話題帳 を切り替え |
| q | 終了 |
| ? | キーの説明 |

## 走らせる

```bash
cd sekai-wadaichou
pip install -e ".[test]"
pytest                       # ロジック単体テスト
sekai-wadaichou              # TUI 起動
```

## 哲学

- 「就活がんばれ!」 とは絶対に言わない (禁止語監査あり)
- 「あなたの志望業界はこれ!」 のような決めつけはしない
- 「世界には今、 こんな話題がある」 という事実を 1 行で出すだけ
- 競争 / streak / 連勝記録 / 点数は出さない
- 1 都市 = 1 カード、 12 都市 = 1 周、 数分で 1 周できる軽さ
