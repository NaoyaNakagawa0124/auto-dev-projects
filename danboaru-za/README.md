# 段ボール座 — danboaru-za

> *Constellatio Cartonensis* — まだ開けていない段ボールは、まだ星のままです。

引っ越したばかりの部屋に積まれた段ボールの山を、夜空にひとつの **星座** として登録するブラウザアプリ。各箱は決定論的な座標と等級を持つ恒星になり、近い星と細い線でつながって独自の **段ボール座** を成します。

「開封」ボタンを押すと、該当する恒星は **超新星爆発** を起こし、夜空からゆっくり消えます。すべての箱を開け終えた瞬間、夜空は再び静かになり、**「完全移住完了」** と公式に記録されます。

これはタスク管理アプリですが、片付けというより **観測** に近い体験を狙いました。

## どう動くか

1. パネルの「新しい星を登録する」に箱の名前（例: `本`、`キッチン用品`、`謎の箱`）を入れて「星に登録」を押す。
2. 名前から決定論的に座標が決まり、夜空に新しい星が現れる。サイズ（小・中・大）は等級になる。
3. 開封したくなったら、星座の星を直接クリックするか、台帳の「開封」ボタンを押す。**超新星爆発** のあと、星はそっと消えます。
4. 全ての箱が開封されると、夜空は再び静かになり「完全移住完了」が記録されます。

## 走らせる

外部依存はサーバーランタイム不要、p5.js だけ CDN から読みます。

```sh
cd danboaru-za
python3 -m http.server 8765
# → http://localhost:8765/
```

`file://` でも一応動きますが、ES モジュールを使っているのでサーバ越しを推奨。

## テスト

純粋ロジック層（ハッシュ・乱数・星配置・最近傍辺・台帳）は Node 標準のテストランナーで走ります。p5.js などのブラウザ依存は触っていません。

```sh
node --test test/core.test.mjs
```

24 ケース通ります。

## 設計メモ

- **Intent 3「こんなのアリ？」と笑わせる** に振り切った。
  - タスクなのに完了が「天体ショー」。
  - 番付やストリークではなく **空が空っぽになるかどうか** が唯一の指標。
  - パネルの肩書きは `DBZ-CATALOGUE 2026.05` で、まるで本物の天文台のように振る舞う。
- 星の位置は箱名の **FNV-1a-ish ハッシュ** から **mulberry32 PRNG** で決定的に生成。
  同じ箱名は常に同じ位置に出る。再現性のためのライブラリは使わずに数十行の純 JS。
- 星座の線は **2-最近傍** の重複削除エッジ。グラフが極端に偏らないので絵的に綺麗。
- 等級平均（小=0.55 / 中=1.0 / 大=1.6）は本物の天文に倣ったわけではなく、UI に出すと **それっぽい** 値が並んで気持ちいいので採用。
- 完了の達成感を「カタログ印字」風に演出。`Logged YYYY-MM-DD · DBZ Catalogue` がさりげなく書かれる。
