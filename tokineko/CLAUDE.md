# tokineko

Python Textual TUI app. Virtual cat companion with time travel mechanics.

## Run
```bash
pip install -e .
tokineko
```

## Test
```bash
python -m pytest tests/ -v
```

## Conventions
- All user-facing text in Japanese
- English variable/function names (snake_case)
- Textual framework for TUI
- JSON file for persistence (~/.tokineko/save.json)
