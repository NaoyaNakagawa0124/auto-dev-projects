# tokineko（時猫）

> ターミナルに住む、時間旅行する猫のコンパニオン 🐱⏳

孤独なリモートワーカーのためのターミナル仮想ペット。ポモドーロタイマーで作業するたびに、あなたの猫が歴史の異なる時代を旅します。

## Features

- **時間旅行する猫** — 先史時代から未来まで、9つの時代を冒険
- **ポモドーロ連動** — 25分の作業セッション完了で猫が次の時代へ
- **美しいTUI** — Textualによるリッチなターミナルインターフェース
- **コレクション要素** — 各時代でアイテムを発見・収集
- **猫のステータス** — 幸福度・エネルギー・好奇心が変化
- **実績システム** — 全時代訪問、全アイテム収集などの実績解除

## Tech Stack

- Python 3.11+
- Textual (TUI framework)
- Rich (terminal formatting)
- JSON file storage

## How to Run

```bash
cd tokineko
pip install -e .
tokineko
```
