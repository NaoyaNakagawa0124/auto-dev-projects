# 流彩 (Ryuusai) — デジタル墨流しアートジェネレータ

墨流し（Suminagashi）の流体物理シミュレーションを使って、美しいデジタルアートを生成するWebアプリ。

## 特徴

- **リアルな流体物理** — マランゴニ対流・ケルビンヘルムホルツ不安定性をベースにした墨の広がり
- **インタラクティブ操作** — クリックで墨を落とし、ドラッグで流れを作り、息を吹きかけるように操作
- **カラーパレット** — 伝統的な和色から現代的な配色まで選択可能
- **高解像度エクスポート** — PNG/SVG形式で書き出し、プリントオンデマンド販売に対応
- **ギャラリーモード** — 作成したアートを一覧表示・管理

## 技術スタック

- **p5.js** — 描画エンジン・物理シミュレーション
- **HTML/CSS/JavaScript** — フロントエンド
- **Web Workers** — 重い物理計算をバックグラウンドで処理
- **LocalStorage** — 作品の保存

## 起動方法

```bash
# ファイルを直接開く、またはローカルサーバーで起動
npx serve .
# ブラウザで http://localhost:3000 を開く
```

## 副業活用

生成したアートは以下のプラットフォームで販売可能：
- SUZURI / Redbubble (プリントオンデマンド)
- BOOTH / Gumroad (デジタルダウンロード)
- ストックフォトサイト (素材販売)
