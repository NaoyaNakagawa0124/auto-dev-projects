# Ryuusai Project

## Overview
Digital suminagashi (Japanese marbling) art generator using fluid physics simulation with p5.js.

## Tech Stack
- p5.js for rendering
- Vanilla JS (no framework)
- CSS with glassmorphism design

## Conventions
- All user-facing text in Japanese
- camelCase for variables/functions
- ES modules where possible
- Mobile-first responsive design
