# Nyantokashite — Implementation Plan

## Phase 1: Core Game Engine
- Grid-based game board (Canvas 2D)
- Player movement (arrow keys / WASD)
- Push mechanics (Sokoban-style)
- Tile types: floor, wall, item, target, player, cat

## Phase 2: Level System & Content
- Level format (2D array with tile codes)
- 20+ handcrafted levels across 4 rooms
- Star rating (moves vs par)
- Level select screen

## Phase 3: Cat Mechanics & Polish
- Cat character that moves around the level
- Cat sometimes blocks paths (must wait or lure)
- Undo/redo system
- Cute pixel-art sprites (CSS/Canvas)
- Animation and particle effects

## Phase 4: UI & Final
- Title screen with cat animation
- Japanese text throughout
- Sound effects (Web Audio)
- localStorage for progress
- Responsive layout
