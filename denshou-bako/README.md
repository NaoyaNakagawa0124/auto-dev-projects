# 伝承箱 (denshou-bako)

> 老後を楽しんでいるシニアが、 数十年ぶんの仕事の知恵を、 一日 1 分ずつ録音して
> 残していく Raspberry Pi の枕元の箱。 10 年後に家族・後輩・会社の次世代が
> 開いたとき、 まだ価値が残っていることを目標にしたアーカイブ装置。

## なぜ

「定年退職した人の頭の中の知恵」 ─ 30 年の現場で培った 部下の叱り方、
予算の切り方、 取引先との距離感、 危機の対処、 採用の見極め — その
ほとんどは、 本人がいなくなった瞬間に消える。 残せる形が無いから。

伝承箱は、 シニア本人が「面倒くさい記録作業」 を一切しないで済むように、
**Raspberry Pi + 大きなボタン + マイク + 小さなスピーカー** だけの
極小ハードウェアで作る。

毎晩同じ場所 (枕元 / 書斎の机 / 居間のテーブル) に置いておく。 一度
ボタンを押すと、 その日の問いが TTS で読み上げられる。 もう一度押すと
録音が始まり、 さらに押すと止まる。 最長 2 分、 1 日 1 問。

3 年続ければ、 1000 個以上の B2B 知恵の音声アーカイブができる。

## 何を残すのか

365 個の問いは 7 つのカテゴリにまたがる:

- **仕事の哲学** — 「何のために働いてきましたか?」
- **ひととの距離** — 「初めて部下を叱った日のことを話してください」
- **お金の感覚** — 「予算が半分になった時、 まず何を切りましたか?」
- **失敗から学んだ** — 「人生で一番高くついた判断は何でしたか?」
- **道具と現場** — 「いまの若い人が手放した、 自分が大事にしていた道具は?」
- **時代の変化** — 「あなたの業界で、 20 年で変わったいちばん大きなことは?」
- **後輩へ** — 「来週入社する新人 1 人にだけ伝えるとしたら?」

問いは決定論的に日付に対応する (1 月 1 日は常に同じ問い)。
答えたくない日は、 ボタンを 2 秒長押しで「今日は飛ばす」 を選べる。

## ハードウェア

- Raspberry Pi 4 (2GB で十分) または Pi Zero 2 W
- USB マイク (Plantronics 等) または ReSpeaker 2-Mics HAT
- パッシブスピーカー (3.5mm 出力 + 小さいスピーカー、 または HDMI モニタ)
- 大きい押しボタン (5 cm 径、 GPIO 17 接続)
- LED 赤 (録音中、 GPIO 23) + 黄 (待機、 GPIO 24)
- 木の小箱 (任意。 100 円ショップでも可)

詳細は [RUNNING.md](RUNNING.md) を参照。

## ソフトウェア

```bash
# Pi 上で
git clone <this> && cd denshou-bako
pip install -e .
denshou play-today     # 今日の問いを声で聞く (録音はしない)
sudo denshou record    # ボタン受付モード — ふつうは systemd で自動起動

# 自分の Mac/PC で archive を眺める
denshou book ~/path/to/recordings > wisdom_book.md
denshou book ~/path/to/recordings --html > wisdom_book.html
```

## 10 年後に開く方が、 本人より得るものが大きい

伝承箱は、 本人が「役に立つ」 と感じるためのアプリではない。
本人が居なくなった後で、 家族・後輩・教え子が再生したときに、
「あの人が、 あの状況をどう考えていたか」 がはっきり残っているための装置。
Intent 6 ── 「10 年後も価値があるか」 だけを評価軸にしている。
