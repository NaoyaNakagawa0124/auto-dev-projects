# 忘れノート (WasureNote) — 心のガベージコレクション

日記の逆。悩みや後悔を書いて、美しく消し去ろう。

## 特徴

- **書いて、消す** — 悩みを書いて「手放す」ボタンで儀式的に消滅
- **パーティクル演出** — 文字が粒子になって散っていく美しいアニメーション
- **3つの消し方** — 燃やす🔥、溶かす💧、風に飛ばす🌬️
- **何も残らない** — データは一切保存されない。これが目的
- **統計なし** — ストリークも、記録も、プレッシャーもない

## Tech Stack

- Vanilla JS / Canvas API / Vite
- 意図的にlocalStorageを使わない

## 実行方法

```bash
npm install
npm run dev
```
