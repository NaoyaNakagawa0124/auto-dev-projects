# 映画道 (Eigamichi) - ビジュアルノベル風映画レコメンダー

A visual-novel style movie recommender. Answer branching questions about your mood, and navigate a story-like tree to discover your perfect film.

## Features

- **分岐選択UI**: ビジュアルノベル風の選択肢で映画を探す
- **気分ベース**: 「今の気分」から始まる直感的な探索
- **映画道の可視化**: たどった経路がツリーとして表示される
- **80作品+**: 豊富な映画データベース
- **マルチエンディング**: 異なる道は異なるジャンル・作品に到達

## How to Run

```bash
open index.html
```

## Tech Stack

- Vanilla JS / HTML5 / CSS3
- Canvas (path visualization)
- No dependencies
