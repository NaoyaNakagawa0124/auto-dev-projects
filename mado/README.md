# 窓 — mado

> 今日の窓を、ひとつ。世界のどこかの街にそっと寄り添う、語学の小さな旅。

`mado` は、語学学習を「覚える」ではなく「眺める」に変える Web アプリです。
画面いっぱいに開いた窓の向こうに、世界のどこかの街の朝・昼・夜がゆっくり広がります。
そこから聞こえてくるかもしれない 3 つの言葉を、無理なく受け取って、気が向いたら手帳にしまっておく — それだけ。

ストリーク、クイズ、ランキング、通知。なにもありません。

## 6 つの窓

| # | 街 | 時刻 | 言語 |
|---|----|------|------|
| 1 | パリ | 夜明け前 | フランス語 |
| 2 | リスボン | 午後三時 | ポルトガル語 |
| 3 | ハノイ | 夕暮れ | ベトナム語 |
| 4 | ストックホルム | 雪の朝 | スウェーデン語 |
| 5 | イスタンブール | 夕焼け | トルコ語 |
| 6 | レイキャビク | 深夜（オーロラ） | アイスランド語 |

それぞれの窓には固有の：
- 空のグラデーション（時刻と緯度をふまえた配色）
- シルエット（屋根、椰子、針葉樹、ドーム、山並み）
- 環境音（Rust が手づくりするドローン＋風＋鳥／虫／鐘）
- 浮遊するもの（埃／雪／蛍／オーロラの粒子）

があります。

## 技術

| レイヤー | 使ったもの |
|---------|------|
| シーン生成 | **Rust → WASM** （`wasm-bindgen`） |
| 音 | Rust が DSP 合成した 14 秒のシームレスループを Web Audio で流す |
| 描画 | Canvas 2D（Rust 側で出した高度マップ・粒子座標を JS で描画） |
| UI | プレーン HTML / CSS / ES Modules — フレームワークなし |
| フォント | Noto Serif JP / Noto Sans JP / Cormorant Garamond |
| 永続化 | `localStorage`（あつめた言葉の手帳） |

WASM バンドルは約 31KB、テーマ全体で外部ランタイムなし。

## 走らせかた

### 必要なもの

- Rust toolchain と `wasm-pack`（再ビルドする場合のみ）
- 任意の HTTP サーバ（`file://` だと WASM の fetch がブロックされます）

### ビルド済みでそのまま開く

`pkg/` には 31KB の `mado_bg.wasm` と JS バインディングが同梱されています。
ローカルで HTTP サーバを立てるだけで動きます：

```sh
python3 -m http.server 8765
# → http://localhost:8765/
```

### Rust から再ビルド

```sh
wasm-pack build --target web --out-dir pkg --release
```

### Rust 側のテスト

```sh
cargo test --lib
```

## ファイル

```
mado/
├── Cargo.toml
├── src/lib.rs           # シーン・粒子・音響合成（Rust）
├── pkg/                 # wasm-pack の出力（同梱済み）
├── data/windows.js      # 6 つの窓のメタ情報と言葉
├── index.html           # マークアップ
├── style.css            # ガラスモーフィズムの静かな UI
├── main.js              # WASM ロード・キャンバス描画・音・手帳
└── README.md
```

## 設計メモ

- **Intent**: そっと寄り添う。使い終わって心拍が下がる体験。
- 通知やストリークを意識的に削った。語学アプリの「がんばらせる」設計と逆。
- シルエット・粒子・音は Rust 側で「シーン番号」だけを引数に決定論的に生成される。
  シーンを切り替えるとアニメーションが連続的に変化する。
- 音は 14 秒のループで、末尾と先頭をクロスフェードしてシームレスに繋いでいる。
