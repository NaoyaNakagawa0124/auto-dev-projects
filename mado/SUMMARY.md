# SUMMARY — mado

## What was built

A quiet, ambient web app where a user opens a "window" to one of six cities — Paris dawn, Lisbon afternoon, Hanoi dusk, Stockholm winter morning, Istanbul sunset, Reykjavik aurora night. The window animates with scene-specific sky gradient, layered silhouettes (rooftops / palms / pines / domes / mountains), particle effects (dust / snow / fireflies / aurora), and a 14-second seamless ambient loop synthesized in Rust. Three travel-friendly phrases for the local language sit gently below. The user can collect phrases into a personal journal — that's all. No streaks, no quizzes.

## Tech decisions

- **Rust + WASM** for everything that benefits from being deterministic + performant: gradient/silhouette parameters, particle simulation, ambient audio synthesis (oscillators + filtered noise + accent envelopes). The WASM bundle is **31 KB** and the JS bindings are 8.5 KB — no external runtime.
- **Canvas 2D** instead of WebGL — sufficient for the visual fidelity we want, and lets us pull data out of WASM (gradient bytes, height maps, particle positions) without GPU plumbing.
- **Web Audio API** consumes a Float32Array generated by Rust and loops it. The Rust synth crossfades head/tail for a clean seamless loop.
- **No framework** — plain ES modules. Boot time is dominated by font loading, not app code.
- **Glassmorphism** + Noto Serif JP / Cormorant Garamond italic for the foreign phrases. The accent color shifts per window so the whole UI breathes with the scene.

## Tests

8 Rust unit tests covering scene count, gradient byte length, silhouette range, particle spawning, audio buffer length & sample range, seamless loop endpoint matching, and that two different scenes produce audibly different audio.

## Potential next steps

1. **More windows**. The Rust scene system is data-driven (per-index match arms); adding Marrakesh, Kyoto, or Buenos Aires is just a new entry plus matching phrase data.
2. **Light pronunciation hints via Web Speech API**. Currently the user reads the romanized line; speaking it out loud (only on tap, never auto) would deepen the moment without breaking the silence.
3. **Sharable postcards**. Render the current canvas + collected phrase to a PNG postcard the user can send. Keeps the "souvenir" feeling.
