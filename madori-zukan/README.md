# 間取り図鑑 (madori-zukan)

> アニメ・漫画の家を「不動産物件」として記録する PWA。視聴履歴をアーカイブに変える。

## これは何？

**間取り図鑑** は、アニメ・漫画に登場する架空の家を、本物の不動産情報のように記録するアーカイブアプリです。

- 野比家・磯野家・野原家・草壁家・竈門家・フォージャー家… 名作の家を **間取り図** で見られる
- 各部屋には「誰がここで何をしたか」「印象的なシーン」が紐づく
- 観たことのある作品にチェックを入れ、お気に入りに登録すると、自分の「アーカイブ坪数」が増えていく
- 全部オフラインで動く PWA。スマホのホーム画面に追加して、いつでも開ける
- データはローカル保存。十年後も自分の視聴史として残る

> 10 年後にこのアーカイブを開いて「ああ、あの頃こんな作品を観てたな」と振り返れるよう作っています。

## 特徴

- **間取り図表示**: 各家を SVG で描画。部屋・壁・玄関・窓・縮尺・方位を含む建築図面風
- **物件カード**: 築年代・延床面積・住所（架空）・住宅様式を不動産チラシ風に
- **部屋詳細**: タップで各部屋にズーム、住人と思い出のシーンを表示
- **アーカイブ統計**: 観た作品数、お気に入り数、累計記録延床面積、世代別構成
- **絞り込み**: 年代 / ジャンル / 住宅様式 / 観了済みのみ
- **オフライン**: Service Worker でフルキャッシュ。電波がなくても閲覧可
- **全 UI 日本語 + 縦長スマホ最適化**

## 動かし方

```bash
cd madori-zukan
python3 -m http.server 8000
# ブラウザで http://localhost:8000 を開く
```

スマホ: 同 LAN 経由でアクセス → 共有メニュー → ホーム画面に追加で PWA インストール。

### テスト

```bash
node --test "tests/*.test.mjs"
```

## 技術スタック

- Vanilla HTML / CSS / JavaScript (ES Modules) — ビルドステップなし
- SVG (手書き) で間取り図描画
- localStorage で永続化
- Service Worker でオフラインキャッシュ
- Web App Manifest でホーム画面インストール対応
