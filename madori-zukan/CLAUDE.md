# madori-zukan — プロジェクトコンベンション

- 全 UI 日本語。明朝系をタイトルに、ゴシック系を本文に使う
- 1 畳 = 1.62㎡ (関東基準)
- 部屋座標系は 10 = 1m。SVG viewBox は dynamic
- カラー: paper #f7f4ee, ink #1c1b1a, plan-line #2b3a4a, room-fill #fdfbf3, accent #c8956d
- データの編集は `data/homes.json` のみ
- localStorage キー: `madori-zukan/state/v1`
- 部屋タイプ: entry / living / dining / kitchen / bedroom / bath / wc / work / garden / corridor / japanese / closet
