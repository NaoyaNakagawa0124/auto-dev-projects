# 論理ルーム (RonriRoom) — 新人脱出パズル

入社初日、なぜかオフィスの部屋に閉じ込められた！論理パズルを解いて脱出せよ！

## 特徴

- **5つの部屋** — それぞれ異なる論理パズル（パターン認識、スイッチ、数列、経路、暗号）
- **新人ストーリー** — 入社1年目の主人公がオフィスを探索
- **学術的背景** — 各パズルはCS/数学の概念に基づく
- **レトロスタイル** — Raylibで描く、すっきりとしたピクセル調2Dグラフィック

## Tech Stack

- C / Raylib
- 2D rendering with Japanese text support

## ビルドと実行

```bash
make
./ronriroom
```

## 操作方法

- **矢印キー / WASD**: 選択・移動
- **Enter / Space**: 決定
- **Escape**: メニューに戻る
