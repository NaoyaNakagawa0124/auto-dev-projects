# RonriRoom Implementation Plan

## Phase 1: Engine Foundation
- Makefile with Raylib linkage
- Window setup, game loop, state machine
- Japanese font rendering (Noto Sans JP embedded or system font)
- Screen management (title, game, results)

## Phase 2: Puzzle Engine
- Puzzle data structures (grid-based, selectable cells)
- Input handling (arrow keys, enter)
- Visual feedback (correct/wrong animations)
- 5 puzzle types implemented

## Phase 3: Game Flow & Polish
- Room transitions with story text
- Score tracking (time, mistakes)
- Completion screen
- All Japanese UI text
