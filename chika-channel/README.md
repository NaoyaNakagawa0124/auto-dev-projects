# 地下チャンネル (chika-channel)

> 駆け出しの YouTuber / TikToker が、 自分の動画を **地下鉄路線図** として
> 設計していく中毒系ブラウザゲーム。 Bun + TypeScript + SVG。

「地下チャンネル」 という呼び方には 2 つの意味があります:
- ニッチで地味だけれど熱量の高い YouTube チャンネル
- 地下鉄 (subway) のように、 1 本 1 本の動画が駅で、 ジャンルや
  サムネの雰囲気が「路線」 で繋がっている

このゲームでは、 あなたは ── 100 人しか登録者がいない無名の人として ──
1 本の動画から始め、 駅 (動画) を増やし、 線 (テーマ) を引いて、
視聴者の流れを最適化していきます。 アルゴリズムは天候のように毎週揺らぎ、
路線図が完成するほど 「視聴者の乗換え」 が連鎖して登録者数が伸びていきます。

## ゲームループ (40 秒で 1 日)

1. **昼 (40 秒)** ── 路線図に視聴者の点が流れ、 上部の登録者カウンタが
   ピコピコ伸びていく。 ただ眺める時間。
2. **夜 (アクションフェーズ)** ── アクションポイント 1〜3 個。
   - 駅を 1 つ作る (動画を 1 本上げる)
   - 線を引く / 線にもう 1 駅を繋ぐ
   - サムネを更新する (vibe を上げる)
   - 駅を消す (削除動画 — penalty 無料、 ただし路線ボーナス減)
3. 次の日へ。

毎週ランダムな **アルゴリズム天候** が来ます:
- 「ゲーム系トラフィック半減」
- 「Vlog 季節来訪 +100%」
- 「サムネ戦争 — vibe 1 の駅 -30%」
- 「アルゴリズム初期化」 ── decay 全 reset

登録者数のチェックポイントで新しいタイプの駅 / 線が解放されます。

## なぜ Bun か

- 一発で TypeScript が動く。 ビルドツール不要。
- `bun test` 込みで開発体験が軽い。
- ローカル http サーバーを 3 行で立てられて、 静的アセットも同時に
  サーブできる。
- アスピリング YouTuber 向けのカジュアルゲームに、 重い React や
  Webpack を持ち込む必要がない。

## 動かす

```bash
cd chika-channel
bun install
bun run dev          # http://localhost:5173 で起動
bun test             # 全テスト
```

## 設計指針

- **時間を忘れる** が唯一の評価軸 (Intent 5)。 1 日 40 秒、 アクション 5 秒、
  チェックポイント解放の通知 ── 1 セッション 20 分で気づくと 30 日経って
  10 万登録者に到達している、 を目指す。
- 「YouTuber の世界をリアルに再現」 はしない。 サムネ vibe や CTR を
  数字ゲーム化して 「直感が育つ」 形に倒す。
- 派手な絵文字 / 通知バナーは控えめに。 地下鉄路線図そのものの美しさで
  保留時間を稼ぐ。
