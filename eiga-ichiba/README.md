# 映画市場 (eiga-ichiba)

> 二人で同じキーボードを囲む、映画オタクのための株式投資ゲーム。Raylib 5.5 + C で動く。

## これは何？

**映画市場** は、映画オタクの 2 人が 1 つのキーボードを共有して遊ぶ、月次の映画株式投資シミュレーションです。

- 12 ヶ月のシーズンで、毎月 1 本の話題作が「上場」する
- 2 人のプレイヤーは交互に、自分の手元資金の何 % をその作品に投資するか決める
- 月末に作品が公開され、興行のリターンが公開される
- 12 ヶ月終了時、ポートフォリオ価値の高い方が勝者

ただの勝ち負けではありません。「この監督なら大ヒットだ」「いや、客層が狭すぎる」と話し合いながら同じ画面を見るのがこのゲームの本体です。

## 特徴

- **2 人パスアンドプレイ**: 1 つの PC を囲んで遊ぶ。Player 1 が入力後 Player 2 に渡す
- **12 本の架空話題作**: ジャンル・予算・監督ランク・期待度を持つカード
- **手描き的なポスター**: Raylib の Polygon と矩形で抽象的なポスターを生成
- **リターンの揺らぎ**: 期待度に基づき 0.2x 〜 4.0x のリターン。サイコロ要素あり
- **興行結果アニメーション**: チャートが伸びる演出
- **シーズン総括**: 投資判断の振り返り画面

## 動かし方

```bash
# 前提: macOS で raylib 5.5 がインストールされていること (brew install raylib)
cd eiga-ichiba
make            # ビルド
./eiga-ichiba   # 実行

# テスト
make test
```

Linux でも pkg-config が raylib を解決できれば動きます。

## 操作

| キー | 操作 |
|---|---|
| ↑ / ↓ | 投資割合を選ぶ (0/25/50/75/100%) |
| Enter | 投資を決定 / 次のステップへ |
| Space | アニメーション/結果を進める |
| R | 新規ゲーム開始 |
| ESC | ゲーム終了 |
