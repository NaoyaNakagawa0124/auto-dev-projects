# ごちそう暦 (gochisou-goyomi)

> 毎日 1 つ「今日はどの国の祭日 → 何を食べる？」を提案する Textual TUI。フーディーの献立決定疲れを治す。

## これは何？

**ごちそう暦** は、毎日外食するフーディーのための、文化暦ベースのごはん提案ツールです。

- 起動すると、今日 (または近い日) に祝祭日を持つ国を 1 つ提示
- その国の伝統食 / 国民食を 1 品おすすめ
- 「お店で食べる」「家で作る」のどちらが現実的かのヒント付き
- 食べた料理を 5 段階で記録 → 自動的にカレンダーと統計に反映
- 一年のうちで何ヶ国分の食卓を旅できたか、その国の祝日と一緒に振り返れる

5/17 を起動した時の例:
> 🇳🇴 **ノルウェー憲法記念日** (Syttende Mai)
> おすすめ: **ホットドッグ・イ・ロンペ** (Pølse i lompe)
> ふんわりした薄焼きパンに茹でソーセージを巻く、当日の屋台の定番。
> 推奨: 🏠 自宅でラップサンドで再現が一番早い

## 特徴

- **80 件以上の文化暦エントリ** — 各国の国民祝日・宗教行事・食の日を網羅
- **±3 日のフォールバック** — 今日に該当がなければ近隣日から拾う
- **家 vs 外** ヒント — 入手難易度 (近所のスーパー / 専門店 / 自宅で再現可) を 3 段階で
- **食卓ログ** — 提案を採用したか、別物を食べたかも記録。5 段階評価
- **統計画面** — 今年の踏破国・連続日数・お気に入り料理ランキング
- **完全日本語TUI** — Textual の整然としたパネルレイアウト、温かみのあるカラー

## 動かし方

```bash
cd gochisou-goyomi
pip install -e .
gochisou-goyomi
```

または `python -m gochisou_goyomi` で起動できます。

## 操作

| キー | 操作 |
|---|---|
| Tab / Shift+Tab | パネル/フォーカス切替 |
| 1 〜 5 | 食べた料理を 5 段階評価 |
| O | 提案通り食べた、を記録 |
| X | 別のものを食べた、を記録 |
| C | カレンダー画面 |
| S | 統計画面 |
| Q | 終了 |
