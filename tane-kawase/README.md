# 種交わせ (tane-kawase)

> 語学を勉強中の二人が、ことばの種を交わす CLI。

英語をやっている友人と、フランス語をやっている自分。 互いに「これ覚えて
おくと使えるよ」 という 5-10 個のことばを、 ひと包の「種」 にして渡す。
受け取った包は、 自分の「畑」 に植えれば、 トピックごとに小さな野菜の
列が育っていく。

```
            ━━━━ あなたの畑 ━━━━

  春菜  ψ ψ ψ ψ ψ          5 種   (1 包 / さくら)
  夏野菜 ◯ ◯ ◯              3 種   (1 包 / たくみ)
  秋穀  ‖ ‖ ‖ ‖ ‖ ‖        6 種   (2 包 / たくみ・さくら)
  冬根  ・                  0 種   (まだ)
  草花  ✿                    1 種   (1 包 / さくら)

  最近もらった: 「秋の慣用句」 from たくみ  —  2026-05-10
  最近送った  : 「夏のスラング」 to  さくら  —  2026-05-12
  これまでに交わした包: 14
```

## Why

- 単語帳アプリは一人で完結してしまう。 でも語学は「誰かと話したい」
  気持ちが原動力。
- メッセージで「これいい表現!」 を送り合うのも、 流れて消える。
- **包む → 渡す → 植える** という小さな儀式があると、 ことばが
  「私たち二人のあいだ」 に残る。

## What

ひと包 (`packet.json`) は:
- 送り主の名前
- 受け取り手の名前
- トピック (春菜 / 夏野菜 / 秋穀 / 冬根 / 草花)
- 包の名前 (例: 「秋の慣用句」)
- 5–10 の種 (語/句 + 例文 + 任意のメモ)
- 任意の手紙 (1-2 行)

ファイルとして書き出し、 LINE / メール / USB / どこ経由でも構わない。
受け取った相手が `tane-kawase plant packet.json` すれば、 畑に列が
1 行追加される。

## Tech

- Python 3.10+
- Rich (色とテキスト)
- argparse
- JSON で包と畑を永続化 (`~/.tane-kawase/field.json`)
- pytest

## Install

```bash
cd tane-kawase
pip install -e .
tane-kawase --help
```

## Quick demo (一人でも見られる)

```bash
tane-kawase demo                  # サンプル畑を生成
tane-kawase field                 # 畑を見る
tane-kawase pack                  # 対話的にひと包作る
tane-kawase send packet.json      # ファイル化
```
