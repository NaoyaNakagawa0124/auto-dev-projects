# NijiLog (虹ログ) - 感情が流れるジェネラティブ日記

あなたの日記が、美しいアートになる。

## 特徴

- **感情の色** — 毎日の気分を色で選ぶ。喜び、悲しみ、怒り、穏やか、すべてが色になる
- **流れるパーティクル** — 日記のエントリーがp5.jsのパーティクルストリームとして描画される
- **途切れない流れ** — 連続記録は美しい虹のリボンに。サボると流れが途切れる
- **色で語る** — 言葉が少なくても、色だけで日記になる。1タップで記録可能
- **あなただけのアート** — 時間とともに、唯一無二のジェネラティブアートが完成する

## Tech Stack

- p5.js (generative art / particle system)
- Vanilla JavaScript
- HTML5 / CSS3
- localStorage (persistence)
- Vite (build)

## How to Run

```bash
npm install
npm run dev
```
