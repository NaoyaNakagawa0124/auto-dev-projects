# PawRank

> Every dog's a winner — but some are more winning than others. A competitive dog stats leaderboard.

## What is PawRank?

PawRank is a Jupyter notebook that turns dog ownership into a competition. Enter your dog's stats, see them ranked against 20+ sample dogs across multiple categories. Beautiful Plotly visualizations, radar charts, and leaderboards.

**Everything is a competition.** Best walker. Most tricks. Top obedience. Cutest face. Overall champion.

## Features

1. **Dog Database** — 20+ sample dogs with realistic stats across 8 categories
2. **Leaderboards** — Ranked tables for every category + overall champion
3. **Radar Charts** — Spider charts comparing any dog's profile
4. **Bar Chart Races** — Visual rankings with animated-style category bars
5. **Add Your Dog** — Input your own dog's stats and see where they rank
6. **Trophy Room** — Summary of winners per category

## Categories

| Category | What it measures | Scale |
|----------|-----------------|-------|
| Activity | Daily walks, exercise hours | 1-100 |
| Tricks | Number of tricks learned | 1-100 |
| Obedience | Recall, sit-stay, leash manners | 1-100 |
| Cuteness | Subjective but scientific* | 1-100 |
| Friendliness | How they are with other dogs/people | 1-100 |
| Bravery | Fearlessness score | 1-100 |
| Intelligence | Problem-solving ability | 1-100 |
| Fluffiness | Coat volume and softness | 1-100 |

*Cuteness is always 100. All dogs are cute.
