# 足跡日記 (ashiato-nikki)

> 日記 を 書こう と して 続かない 人 の、 ボタン 1 タップ で 残す ブラウザ 拡張機能。

毎日 何か 書く 必要 は ない。 何か 残したい と 思った 時 だけ、 ブラウザ の ツールバー の 足跡 アイコン を 押す。 開いて いる ページ の URL と タイトル が、 自動 で 今日 の 「足跡」 に 追加 される。 任意 で 1 行 メモ も 書ける。 1 タップ で 終わり。

streak も 「N 日 連続!」 も 「今日 まだ 書いて ない です よ」 も 出ない。 ただ 残す だけ。

## 解決 する 痛み

Quora や Yahoo 知恵袋 に 永遠 に 投稿 され 続ける 「日記 が 続きません」 の 9 割 は、 以下 の どれか:

1. 書く 内容 を 考える の が 重い
2. 「毎日」 が プレッシャー に なる
3. 過去 を 読み返したい 時、 どこ に あるか わからない

足跡日記 は その 3 つ を 解決 する:

1. **内容 を 考えない** — タイトル と URL は 自動、 メモ 欄 は 空白 で 良い
2. **「毎日」 プレッシャー を 出さない** — streak / 連勝 / 「書いて ない 日 です よ」 全部 無し
3. **検索 と タイムライン** — オプション 画面 で 月 ごと に 過去 を 振り返れる、 JSON / Markdown で エクスポート

## 走らせる

```bash
cd ashiato-nikki
npm install
npm test                # 30+ ユニット テスト (Chrome 不要)
```

Chrome / Edge / Brave で `chrome://extensions` を 開き、 デベロッパー モード を ON、 「パッケージ化されていない拡張機能を読み込む」 で `ashiato-nikki/` フォルダ を 選択。

ツールバー の 足跡 アイコン を タップ する と、 ポップアップ が 開く。 今 の ページ を 残す ボタン を 1 タップ、 任意 で メモ を 書いて 完了。
