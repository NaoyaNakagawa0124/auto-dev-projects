# Tsumugu Project

## Overview
Discord bot + CLI that maps Japan's 72 microseasons to craft suggestions.

## Tech Stack
- Node.js, discord.js v14, dayjs
- No database — all data in JS modules

## Conventions
- All user-facing text in Japanese
- camelCase variables/functions
- ES modules (type: module in package.json)
- Color output in CLI using ANSI codes
