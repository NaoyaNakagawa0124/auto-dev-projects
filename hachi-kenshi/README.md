# 鉢検視 (hachi-kenshi)

> 枯らした 観葉 植物 の 死因 を 推理 する Rust + WASM ノワール 法医 ゲーム。

ある日 ふと 気づく と、 部屋 の 観葉 植物 が また 1 つ 枯れて いる。 容疑者 (死因) は 複数 — 過湿、 乾燥、 根腐れ、 光 不足、 寒さ、 害虫、 肥料 焼け。 あなた は 鉢 検視 官。 12 件 の 事件 を 「証拠 → 容疑者 → 起訴」 の 順 で 解く タイム アタック 推理 ゲーム。

```
─ 鉢 検視 録  事件 #04 ────────────────────────────────
  被害 者:  ポトス (Epipremnum aureum)        享年: 8 ヶ月
  発見 日:  2026-05-15  室温: 19°C  湿度: 32%
  場所  :  北 向き 窓辺、 暖房 直撃

  ─ 物的 証拠 ─
   ▣ 葉 全体 が 黄色 く 変色、 茎 の 付け 根 は ふやけて いる
   ▣ 鉢底 から 水 が 滲み 出て いる
   ▣ 直近 2 週間 の 水やり 履歴 — 3 日 おき
   ▣ 土 表面 に 白い カビ
   ▣ 害虫 痕跡 なし、 根 を 軽く 触れる と 黒 ずん で 崩れる

  ─ 容疑 者 (5 名) ─
   1. 乾燥 死  ··  「私 が 犯人 なら 葉 は カラ カラ の はず」
   2. 過湿 死  ··  「鉢 底 の 水 と カビ は 私 を 示す」
   3. 寒 さ    ··  「19°C は 犯行 ライン に 届か ない」
   4. 害虫    ··  「痕跡 なし、 立証 不能」
   5. 肥料 焼け ··  「葉 の 縁 が 焦げ茶 で ない なら 私 で は ない」

   起訴 する 容疑 者 を 選んで ください >  _
─────────────────────────────────────────────────────
```

## なぜ Rust + WASM か

- 各 事件 の 証拠 ↔ 容疑者 ロジック を 純 Rust で 構造 化、 JS と シーム レス に やり取り
- 1 ms 単位 の タイム 計測、 PB 更新 を localStorage で 保存
- ブラウザ で 即 動く、 セット アップ ゼロ、 オフライン で 完結

## 機能

- **12 件 の 事件** — 観葉 / サボテン / ハーブ / 多肉、 4 章 × 3 件
- **死因 7 種** — 乾燥死 / 過湿死 / 根腐れ / 光 不足 / 寒さ / 害虫 / 肥料 焼け
- **証拠 解釈** — 各 容疑者 に 「私 で は ない」 ろう か な 弁明 が ある、 矛盾 を 1 つ ずつ 潰す
- **起訴 → 判決** — 正解 で 「有罪」 黒板、 誤起訴 で 「不起訴」 案件 と なり 減点
- **章 ロック** — 前 章 を 6 件 中 4 件 以上 解く と 解放
- **タイム アタック** — 完走 タイム を 記録、 PB と 比較
- **正解 解説** — 正解 後 に 「根 が 黒く 崩れる の は 過湿 ↔ 根腐れ の 連鎖」 等 の 観葉 植物 ケア 豆 知識

## 起動

```bash
cd hachi-kenshi
cargo test                                      # 純ロジック テスト
wasm-pack build --target web --out-dir www/pkg
cd www && python3 -m http.server 8000
# → http://localhost:8000
```

## テスト

```bash
cargo test
```

純ロジック (cases / verdict / score) を 全部 カバー。 UI は 手動 smoke。
