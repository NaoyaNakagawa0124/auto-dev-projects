# アイドルヒストリア (Idol Historia) - 日本アイドル文化50年の軌跡

An interactive Jupyter notebook visualizing 50+ years of Japanese idol culture history — from the Showa era to today.

## Features

- **年代別タイムライン**: Major milestones from 1970s to 2020s in beautiful Plotly charts
- **時代の探検**: Deep-dive into each era's defining idols, songs, and cultural shifts
- **デビュー分析**: Patterns of idol debuts across decades
- **ジャンル進化**: How idol music and aesthetics evolved over time
- **今日の推し史**: What happened "on this day" in idol history
- **推し文化データ**: Fan culture statistics and trends visualization

## How to Run

```bash
pip install jupyter plotly pandas matplotlib
jupyter notebook idolhistoria.ipynb
```

## Tech Stack

- Python / Jupyter Notebook
- Plotly (interactive charts)
- Pandas (data processing)
- Matplotlib (static charts)
