# ShotList 2035

A 2035-themed Electron app acting as an AI Director of Photography. Analyzes 2026 trending visual styles and generates personalized shot lists with difficulty ratings, gear suggestions, viral potential scores, and AI notes.

## Features

- 10 trending visual styles (Cinematic, Neon, Bold Color, Candid, Retro, etc.)
- 26 shot templates with descriptions, difficulty, and gear requirements
- Style filtering and skill-level matching (Beginner → Pro)
- Viral potential scoring per shot based on trend data
- AI Director notes with strategic advice
- Shot list summary with style mix, difficulty breakdown, and gear list

## Run

```bash
open src/index.html
# or with Electron: npm install && npm start
```

## Test

```bash
node tests/test.js
```
