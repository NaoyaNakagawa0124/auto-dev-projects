# 🎫 tiketto — チケットコレクション

ライブ・コンサートの参戦チケットを美しくコレクションするElectronデスクトップアプリ。DevTools風ダークUIで、統計・タイムライン表示付き。

## 特徴

- **チケットスタブUI** — 半券風デザインのカードでコレクション表示
- **統計ダッシュボード** — 参戦数、アーティストランキング、月別ヒートマップ
- **タイムライン** — 年別の参戦履歴を時系列で閲覧
- **検索・ソート** — アーティスト名、会場名で即座にフィルタリング
- **カラーコード** — チケットごとに色分けして視覚的に管理
- **5段階評価** — 各公演を★で評価
- **データ書出/読込** — JSONバックアップ・リストア対応

## 技術スタック

- **Electron** — クロスプラットフォームデスクトップアプリ
- **Vanilla HTML/CSS/JS** — フレームワーク不要の軽量実装
- **JSON File Store** — SQLite不要のシンプルな永続化

## 起動方法

```bash
cd tiketto
npm install
npm start
```

## テスト

```bash
node tests/run.js
```
