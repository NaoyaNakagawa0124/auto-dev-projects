# 文豪暦 (bungou-reki)

> 読書家のためのカードバトル文学暦。今日に縁ある文豪を召喚し、コレクションを育て、毎日の対戦に勝利せよ。

## これは何？

**文豪暦** は、文学史上の「この日」の出来事をカードに変換するデスクトップ対戦アプリです。

- 毎日、その日に誕生・没年・代表作刊行があった作家が3名「召喚」され、1名だけコレクションに加えられます
- 作家には実在の経歴に基づく4つのステータス（**文才・多作・影響・寿命**）があります
- 集めた作家でデッキを組み、「本日の挑戦者」と3ラウンドの対戦に挑みます
- 実際に読んだ本を「読書ログ」に記録すると、対応する作家にXPが入り強化されます
- 蓄積される一年の文学暦をめくる楽しみは、年間100冊読む読書家のための喜びです

## 特徴

- **実在の文学暦**: 50名以上の日本・世界の文豪と、彼らの生没年・代表作刊行日を実データから収録
- **古書館の意匠**: セピア＋墨色＋金箔のパーチメント調デザイン。明朝体のJSフォントスタック
- **オフライン完結**: Tauriデスクトップアプリ。データはローカルJSONに永続化
- **読書連動成長**: コレクション中の作家の作品を読むたび、その作家にXPと「読み込み」称号が付与
- **永久カレンダー**: 366日分の文学暦をいつでも閲覧。未来の日付も先取り可能

## 技術スタック

- **Tauri 2.0** (Rust 1.94 + Web frontend)
- **Frontend**: Vanilla HTML/CSS/JavaScript（ビルドステップなし）
- **Backend**: Rust（chrono, serde, tauri）
- **永続化**: AppData配下のJSONファイル
- **テスト**: Rust標準テスト + Node.js node:test

## 動かし方

### ブラウザ単体で試す（推奨：素早い体験）

```bash
cd src
python3 -m http.server 8000
# http://localhost:8000 をブラウザで開く
```

ブラウザ版は localStorage に保存されます。データはRust版と互換です。

### Tauriデスクトップアプリとして起動する

```bash
# 初回のみ
cargo install tauri-cli --version "^2.0"

cd src-tauri
cargo tauri dev    # 開発モード
cargo tauri build  # リリースビルド
```

### テスト

```bash
cd src-tauri
cargo test           # Rust 戦闘ロジック・データバリデーション

cd ..
node --test tests/  # フロントエンドロジック
```
