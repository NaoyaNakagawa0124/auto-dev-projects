# 休符 (kyuufu)

> 効率厨のゲーマー が、 深夜 2 時に「あと 1 勝だけ」 を言わなくても良くなる Discord bot。

「休符 (きゅうふ)」 は楽譜の中の静かな間、 意図的な無音。 次の音を意味あるものにするための、 大切な沈黙。

このボットは「もう寝ろ」 「効率悪い」 とは絶対 言いません。 効率厨が一番嫌うのは命令されることです。 だから 1 行だけ、 やさしく許可を出します。

> 「今日 の 進捗 は、 もう 十分 です。 ボス は あなたの 睡眠 を 待って くれます。」

## 何ができるか

| スラッシュコマンド | 動作 |
|------------------|------|
| `/yasumi` | 「今日 は ここまで」 と背中を押す 1 行 (40 種類の中からランダム) |
| `/yamedoki <時間>` | あと N 時間でやめたい時の、 自然な区切りの提案 |
| `/kyou` | 今日 (世界のどこか) で起きている文化イベント、 「ゲーム を 閉じて 30 秒、 これに 思いを 馳せて みませんか」 |
| `/oyasumi-time <hh:mm>` | 毎日 hh:mm に、 自動で /yasumi の 1 行を、 設定したチャンネルに投稿させる |

## 哲学

- 命令しない (「寝ろ」 「やめろ」 「もう遅い」 は使わない)
- 数字で煽らない (「あと 1 時間で 1 日 24 時間ですよ!」 などは禁止)
- 比較しない (「他の人はもう寝てる」 など禁止)
- 効率厨を否定しない (「効率を求めるあなたは素晴らしい」 ではなく、 ただ事実を 1 行)
- 1 メッセージ = 60 文字以内、 句点で終わる、 押し付けない
- streak も連勝も「N 日連続でやすんだ!」 もない

## 走らせる

```bash
cd kyuufu
pip install -e ".[test]"

# テスト (Discord 不要、 純ロジック)
pytest

# 実機テスト
export DISCORD_TOKEN="..."   # https://discord.com/developers/applications で取得
kyuufu
```

招待 URL は Discord デベロッパーポータルで生成。 必要なスコープは `bot` + `applications.commands`、 必要な権限は `Send Messages` のみ。
