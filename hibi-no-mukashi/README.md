# 日々の昔 (hibi-no-mukashi)

> 子育てで疲れたパパ / ママが、 子供が寝た後の 3 分間に開く、 静かな歴史の小窓。
> Rust + WASM の超軽量シングルページ。

「今日と同じ日付に、 誰かの台所で、 誰かの庭で、 誰かの机で、
どんな小さなことが起きていただろう」 ── そんな、 教科書に載らない
ちいさな歴史を、 1 日 1 つ、 そっと差し出すアプリ。

歴史アプリにありがちな:
- 大事件・大発明・有名人 ✕ → ぜんぶ採用しない
- リスト形式 ✕ → 1 日 1 つだけ
- 通知・連続記録ゲーム ✕ → 開きたい時だけ開けばいい

そういうものを全部やめて、 ただ静かに、 1 つ。

```
─────────────────────────────────────────────────

                   2026 年 5 月 17 日
                       日々の昔

           1923 年、 5 月 17 日。 ある家の縁側で、
           茉莉花が今年はじめて咲いた。 八歳の娘が
           「お母さん、 ちっちゃい星みたい」 と言った。
           それを母は、 自分の手紙の余白に書きとめた。

                       (motif: 花)

           [今夜の思い]
           > [____________________________________]

           ◀ 5/16     5/18 ▶

─────────────────────────────────────────────────
```

## なぜ Rust + WASM

- オフライン完結 (1 度開けば、 機内モードでも動く)
- 起動が一瞬。 親の 3 分を 0.1 秒も削らない
- 365 個のヴィネットを WASM バイナリに事前焼き込み、 サーバー不要
- 「もう一回開きたい」 心地よさのために、 速さは妥協しない

## 動かす

```bash
cd hibi-no-mukashi
wasm-pack build --target web --release
python3 -m http.server -d web 8080
# http://localhost:8080
```

テスト:
```bash
cargo test
```

## 設計指針

- **完璧でなくていい** ── 365 日全部を埋める必要はない。 ない日は近い日の話で代用、 「近い日の話」 と素直に書く
- **真実性より静けさ** ── すべての vignette は史実か、 歴史的雰囲気に揺るぎなく忠実な創作。 大切なのは事実より、 開いた時に心拍が下がること
- **声に出して読める** ── TTS で読み上げても自然な日本語、 漢字は控えめ
- **干渉しない** ── 通知なし、 streak なし、 共有ボタンなし。 ただ開く
