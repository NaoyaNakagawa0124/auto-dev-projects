# キッズログ (KidLog) — ターミナルで育児記録

忙しい親のための、SNS風ターミナル育児記録アプリ。

## 特徴

- **30秒で記録** — ターミナルからサッと入力。忙しくても大丈夫
- **SNS風タイムライン** — 記録がソーシャルフィード風に表示される
- **リアクション** — 記録に絵文字リアクションを付けられる
- **カテゴリ分類** — 運動、言葉、食事、睡眠、遊び、初めてのこと
- **連続記録** — ストリーク追跡でモチベーション維持
- **統計** — 月別・カテゴリ別の記録数を可視化

## インストールと実行

```bash
pip install textual
python3 src/app.py
```

## 操作

- `n` — 新しい記録を追加
- `↑/↓` — タイムラインをスクロール
- `r` — リアクションを追加
- `s` — 統計を表示
- `q` — 終了
