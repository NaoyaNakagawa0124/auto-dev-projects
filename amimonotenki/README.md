# あみもの天気 (AmimonoTenki) — 気温で編むマフラー

あなたの街の1年間の気温データから、世界にひとつだけの編み物パターンを生成します。

## 特徴

- **気温マフラー** — 1日1段。その日の気温が色になる
- **リアル天気データ** — OpenMeteo APIから過去1年の気温を取得
- **美しいビジュアル** — p5.jsで生成されるグラデーションパターン
- **印刷対応** — パターンを印刷して実際に編める
- **都市検索** — 世界中の都市に対応

## Tech Stack

- p5.js (generative art)
- OpenMeteo API (free, no key required)
- Vite + Vanilla JS
- HTML5 / CSS3

## 実行方法

```bash
npm install
npm run dev
```
