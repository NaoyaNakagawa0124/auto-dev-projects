# ClapBoard 🎬

A Chrome extension that turns your workday into a movie production. Tasks are scenes, projects are productions, retries are takes. Action starts the timer, Cut stops it, That's a Wrap ends the day.

## Features

- 8 genre categories (dev, meeting, design, writing, review, admin, research, break)
- Scene timer with take counting (retakes when you restart a task)
- Productions (projects) with scene/minute tracking
- "Dailies" — daily review of all scenes shot
- Career stats and Director rank (Intern → Legendary Director)
- Director quotes for motivation

## Install

Load as unpacked at `chrome://extensions`, or open `popup.html`.

## Test
