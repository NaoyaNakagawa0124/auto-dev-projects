# 息吹 (IkiFuku) — 呼吸がアートになる5分間

忙しい毎日に、たった5分の呼吸で世界にひとつだけのアートを。

## 特徴

- **呼吸ガイド** — 吸って・止めて・吐いての3ステップ呼吸法
- **ジェネラティブアート** — 呼吸に合わせてキャンバスにリアルタイム描画
- **5分で完成** — セッション終了後、あなただけのアートが完成
- **スクリーンショット映え** — SNSでシェアしたくなる美しいビジュアル

## Tech Stack

- Vanilla JS / Canvas API
- Vite (build)
- Glassmorphism UI

## 実行方法

```bash
npm install
npm run dev
```
