# BlindBite

The anti-DoorDash Chrome extension. Cure decision fatigue with one mystery restaurant pick. No reviews, no ratings, no photos — just trust the universe.

## Features

- One mystery restaurant per click (12 cuisines, 60 restaurants)
- Supply chain trivia: learn where your food's ingredients come from
- Trust Score: tracks how often you go with the pick vs re-roll
- 5 trust levels: Control Freak → Blind Faith
- Streak tracking for consecutive trusted picks
- Detects food delivery sites (DoorDash, UberEats, Yelp, etc.)

## Install

Load as unpacked extension at `chrome://extensions`, or open `popup.html` directly.
